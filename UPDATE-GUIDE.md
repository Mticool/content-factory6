# Обновление конструктора

> ⚠️ Главное правило: обновляй скиллы, НЕ трогай свои данные.

## Быстрое обновление (2 минуты)

1. Открой GitHub → [конструктор](https://github.com/Mticool/content-factory)
2. Нажми зелёную кнопку **Code** → **Download ZIP**
3. Файл скачается в папку **Загрузки** (Downloads)
4. **Двойной клик** по `content-factory-main.zip` → распакуется папка
5. Открой **Терминал** (Mac: Программы → Утилиты → Терминал)
6. **Сделай бэкап** (на всякий случай):

```bash
cp -r ~/content-factory ~/content-factory-backup
```

7. Набери:

```bash
cd ~/content-factory
./update.sh ~/Downloads/content-factory-main
```

Готово! Скрипт сам обновит скиллы и не тронет твои данные (brand, learning, memory).

> 🛟 Если что-то пошло не так — у тебя есть полная копия в `content-factory-backup`.

> 💡 **Рекомендация:** сделай себе напоминание проверять обновления раз в неделю. Мы регулярно добавляем новые скиллы и улучшаем существующие.

## Что обновлять (от нас → к тебе)

| Что | Откуда | Куда |
|-----|--------|------|
| Скиллы | `skills/` | Копируй поверх своих `skills/` |
| CLAUDE.md | Корень | Копируй (если хочешь новый роутинг) |

## Что НЕ трогать (твои данные)

| Папка | Что там | Почему нельзя |
|-------|---------|---------------|
| `brand/` | Твой голос, аудитория, стиль | Потеряешь настройки |
| `learning/` | Что сработало / не сработало | Потеряешь обучение |
| `memory/` | Дневные логи работы | Потеряешь историю |
| `MEMORY.md` | Долгосрочная память | Потеряешь все решения |
| `projects/` | Твои проекты | Потеряешь весь контент |

## Обновление одной командой

```bash
# 1. Скачай свежую версию с GitHub → Code → Download ZIP → распакуй

# 2. Запусти скрипт обновления (он сам сделает бэкап и обновит только скиллы)
./update.sh ~/Downloads/content-factory-main
```

Скрипт `update.sh` лежит в корне конструктора. Он:
- ✅ Копирует только скиллы (новые методики)
- ✅ Делает бэкап старых скиллов
- ✅ НЕ трогает brand/, learning/, memory/, MEMORY.md, projects/
- ✅ Перезапускает OpenClaw (если нужно)

### Ручное обновление (если скрипт не работает)

```bash
# Claude Code — скопируй ТОЛЬКО skills/
cp -r новая-версия/claudecode-content-factory/skills/* твоя-папка/skills/

# OpenClaw — скопируй ТОЛЬКО skills/
cp -r новая-версия/openclaw-content-factory/skills/* ~/.openclaw/workspace/skills/
openclaw gateway restart
```

## Что если случайно перезаписал?

Если скопировал всё поверх и потерял brand/learning/memory:

1. **brand/** — напиши агенту `интервью` — перенастроит голос за 10 минут
2. **learning/** — начнёт копиться заново (давай обратную связь: `это сработало` / `это не сработало`)
3. **memory/** — к сожалению, не восстановить. Но агент быстро наберёт новый контекст

> 💡 **Совет:** перед обновлением сделай бэкап: `cp -r твоя-папка твоя-папка-backup`
