# Быстрый старт: от скачивания до первого поста

> **5 шагов — 15 минут — первый пост готов**

---

## Шаг 1. Подготовка (3 минуты)

Тебе нужны две вещи:

### 1. Подписка Claude Max

Если ещё нет — оформи на [claude.ai/pricing](https://claude.ai/pricing) ($100/мес).

### 2. Установка Claude Code

Открой **Терминал** и набери:

```bash
npm install -g @anthropic-ai/claude-code
```

> 💡 **Где найти Терминал?**
> - **Mac:** Finder → Программы → Утилиты → Терминал
> - **Windows:** Поиск → «PowerShell» или «Terminal»

Если ругается на права доступа:
```bash
sudo npm install -g @anthropic-ai/claude-code
```

> 💡 **Нет npm?** Сначала установи Node.js с [nodejs.org](https://nodejs.org) (нажми кнопку LTS, установи, перезапусти Терминал).

---

## Шаг 2. Скачай конструктор (1 минута)

### Через GitHub Template (рекомендуется):

1. Открой конструктор: https://github.com/Mticool/content-factory
2. Нажми зелёную кнопку **"Use this template"** → **"Create a new repository"**
3. Назови как хочешь (например `my-content-factory`), нажми **Create**
4. Склонируй СВОЮ копию:
```bash
git clone https://github.com/ТВОЙ_ЮЗЕРНЕЙМ/my-content-factory.git
cd my-content-factory
```

### Или через ZIP:
1. Открой https://github.com/Mticool/content-factory → **Code** → **Download ZIP**
2. Распакуй
3. Открой Терминал в этой папке:
```bash
cd путь/к/content-factory
```

---

## Шаг 3. Создай свой проект (1 минута)

```bash
cp -r projects/_template projects/моё-имя
```

Замени `моё-имя` на своё — латиницей, без пробелов:

```bash
# Примеры:
cp -r projects/_template projects/elena
cp -r projects/_template projects/dr-petrov
cp -r projects/_template projects/coach-anna
```

---

## Шаг 4. Запусти и настрой (10 минут)

```bash
claude
```

При первом запуске — пройди авторизацию (следуй подсказкам на экране).

Когда откроется чат, напиши:

```
начать
```

ИИ задаст 3 простых вопроса:

| # | Вопрос | Пример ответа |
|---|--------|---------------|
| 1 | **Кто ты?** | «Я психолог, 8 лет практики. Специализируюсь на тревожности и панических атаках» |
| 2 | **Кому помогаешь?** | «Женщины 30-45 лет, после стресса или родов. Приходят с паническими атаками» |
| 3 | **Как общаешься?** | «На Вы, тепло, без официоза. Юмор мягкий, сарказм — нет» |

Отвечай как есть — не надо думать над формулировками. Система разберётся.

После ответов ИИ:
- ✅ Сохранит твой профиль
- ✅ Настроит голос
- ✅ Сгенерирует 3 поста

---

## Шаг 5. Сгенерируй контент! 🎉

Теперь можешь генерировать контент одним словом:

```
threads 5
```
→ Получишь 5 готовых постов для Threads

```
youtube script про [тему]
```
→ Получишь сценарий для YouTube-видео

```
carousel про [тему]
```
→ Получишь структуру карусели для Instagram

## Что дальше?

### Углуби настройку (по желанию)

Напиши `интервью` — полное интервью за 20-30 минут. После него контент будет ещё точнее попадать в твой стиль.

### Обучай систему

После публикации — давай обратную связь:

- Пост зашёл → напиши `это сработало`
- Пост провалился → напиши `это не сработало`

С каждым разом контент будет всё точнее.

### Все команды

| Что написать | Что получишь |
|--------------|-------------|
| `threads 5` | 5 постов для Threads |
| `youtube script [тема]` | Сценарий для YouTube |
| `carousel [тема]` | Карусель для Instagram |
| `reels [тема]` | Идея для Reels |
| `telegram [тема]` | Пост для Telegram |
| `критика` | Разбор контента от 3 критиков |
| `интервью` | Глубокая настройка голоса |
| `это сработало` | Запомнить удачный формат |
| `это не сработало` | Запомнить что не работает |

## Частые вопросы

**Я не программист — справлюсь?**
Да! Нужно только набрать несколько команд в Терминале. После установки работаешь с ИИ обычным текстом — как в мессенджере.

**Посты не похожи на меня — что делать?**
Скажи `интервью` и пройди полную настройку. Или добавь примеры своих постов: скопируй удачный пост и скажи `это сработало`.

**Как обновить конструктор?**
Скачай свежую версию: https://github.com/Mticool/content-factory → Code → Download ZIP. Скопируй `skills/` поверх своих (brand/ и learning/ НЕ трогай). Подробнее — см. [UPDATE.md](UPDATE.md).

**Что если сломаю?**
Ничего страшного. Твои данные (brand, learning) в безопасности — они хранятся отдельно от системных файлов.
