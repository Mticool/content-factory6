## Как устроен конструктор

```
Пользователь говорит: "напиши пост для Threads"
                          │
                          ▼
              ┌──────────────────────┐
              │   CLAUDE.md (роутер) │ ← точка входа проекта
              │   projects/{name}/   │
              └──────┬───────────────┘
                     │ определяет формат
                     ▼
    ┌────────────────────────────────────────┐
    │          МОДУЛЬ (modules/)             │
    │  threads / youtube / reels / carousel  │
    │  telegram / youtube-shorts             │
    │  Настройки: Notion, вебхуки, шаблоны   │
    └────────┬──────────────┬───────────────┘
             │              │
     основной скилл    доп. скиллы
             │              │
             ▼              ▼
    ┌──────────────┐  ┌──────────────────┐
    │ ФОРМАТ-СКИЛЛ │  │ УСИЛИТЕЛИ        │
    │ threads/     │  │ selling-meanings/ │
    │ youtube/     │  │ storytelling/     │
    │ reels/       │  │ copywriting/      │
    │ carousel/    │  │ editing/          │
    │ telegram/    │  │ headlines/        │
    └──────────────┘  └──────────────────┘
             │              │
             └──────┬───────┘
                    ▼
           ┌───────────────┐
           │   BRAND (/)   │ ← голос, аудитория, ДНК
           │ profile.md    │
           │ personal-dna  │
           │ voice-style   │
           │ audience.md   │
           └───────────────┘
```

## 30 скиллов: что есть и зачем

### 🔵 ФОРМАТЫ — генерация контента под площадку

Основные скиллы. Каждый отвечает за один формат.

| Скилл | Что делает | Размер | Подключает |
|-------|-----------|--------|------------|
| `threads/` | Посты для Threads | 128K | + selling-meanings, storytelling |
| `youtube/` | Скрипты для YouTube (длинные) | 208K | + storytelling, selling-meanings |
| `reels/` | Reels / Shorts / TikTok | 140K | + selling-meanings, nano-banana, heygen |
| `carousel/` | Instagram карусели | 80K | + nano-banana |
| `telegram/` | Посты для Telegram-каналов | 28K | + selling-meanings, storytelling |

**Когда загружается:** пользователь просит контент → CLAUDE.md роутит → загружается формат-скилл + brand/ + learning/

---

### 🟢 УСИЛИТЕЛИ — методологии для улучшения контента

Эти скиллы НЕ генерируют контент сами. Они подключаются к формат-скиллам.

| Скилл | Что делает | Размер | Кто использует |
|-------|-----------|--------|----------------|
| `storytelling/` | Истории, кейсы, 3-актная структура, 18 триггеров | 208K | threads, youtube, telegram |
| `selling-meanings/` | Продающие смыслы, CTA, формулы (PAS, AIDA), Хормози | 256K | все форматы (продающий контент) |
| `copywriting/` | Метод скользкой горки Халилова | 32K | все (текстовые) |
| `editing/` | Инфостиль Ильяхова + убеждение | 32K | финальная редактура |
| `headlines/` | Заголовки и хуки, формулы внимания | 32K | все форматы |

**Когда загружается:** автоматически вместе с формат-скиллом (см. таблицу "Подключает" выше) или по запросу ("сделай продающим")

---

### 🟡 СТРАТЕГИЯ — понять себя, клиента, продукт

| Скилл | Что делает | Размер | Результат |
|-------|-----------|--------|-----------|
| `personal-unpacking/` | Распаковка личности эксперта (8 блоков) | 36K | → `brand/personal-dna.md` |
| `customer-research/` | Распаковка смыслов ЦА, 8 столбиков (Сердитов) | 144K | → обещания, аргументы, буллеты |
| `offer/` | Ядро оффера: формулы, психология, Хормози, Тимочко | 328K | → позиционирование, tripwire, ценообразование |
| `launch/` | Система запусков PLF × Тимочко | 148K | → прогрев, контент-план запуска |

**Когда загружается:** по запросу пользователя ("распакуй меня", "сделай оффер", "план запуска")

> ⚠️ `offer-core/` — дубль `offer/` (те же файлы в другой структуре). Использовать `offer/`.

## Что читать агенту

**Порядок загрузки файлов:**

```
1. ЭТОТ ФАЙЛ (SKILLS-MAP.md) → понять архитектуру
2. projects/{name}/CLAUDE.md  → понять конкретный проект
3. brand/*                    → понять эксперта
4. skills/{нужный}/SKILL.md   → методика формата
5. learning/*                 → что работало раньше
```

---

## Быстрая навигация для агента

**"Мне нужно написать пост"** → читай `skills/{формат}/SKILL.md` + `brand/`
**"Мне нужно распаковать себя"** → читай `skills/personal-unpacking/SKILL.md`
**"Мне нужен оффер"** → читай `skills/offer/SKILL.md`
**"Мне нужна картинка"** → читай `skills/nano-banana/SKILL.md`
**"Что тут вообще есть?"** → читай этот файл
