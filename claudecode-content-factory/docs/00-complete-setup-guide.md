# Полная инструкция: от нуля до первого поста

> Пошаговый гайд для тех, кто никогда не работал с Claude Code.
> Время: 20-30 минут на установку + 30 минут на первый пост.

---

## Что тебе понадобится

- **Компьютер** (Mac, Windows или Linux)
- **Интернет**
- **Банковская карта** (для подписки Claude Max)

---

## Шаг 2: Установи Node.js

Claude Code требует Node.js — это программа, которая позволяет запускать код на JavaScript.

### Для Mac

#### Вариант A: Через Homebrew (рекомендуется)

1. Открой **Terminal** (Finder → Программы → Утилиты → Terminal)

📸 *[скриншот: как найти Terminal]*

2. Скопируй и вставь эту команду, нажми Enter:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

3. Дождись установки Homebrew (может занять 5-10 минут)

4. Установи Node.js:

```bash
brew install node
```

#### Вариант B: Скачать установщик

1. Перейди на **https://nodejs.org**
2. Скачай версию **LTS** (кнопка слева)
3. Открой скачанный файл
4. Следуй инструкциям установщика (Next → Next → Install)

📸 *[скриншот: страница nodejs.org с кнопкой LTS]*

### Для Windows

1. Перейди на **https://nodejs.org**
2. Скачай версию **LTS** (кнопка слева)
3. Открой скачанный файл (.msi)
4. Следуй инструкциям установщика:
   - Accept license agreement
   - Next → Next → Install
   - Поставь галочку "Automatically install necessary tools"
5. Дождись завершения

📸 *[скриншот: установщик Node.js на Windows]*

### Проверка установки

После установки проверь, что всё работает:

1. Открой Terminal (Mac) или PowerShell (Windows)
2. Введи:

```bash
node --version
```

3. Если видишь номер версии (например, `v20.10.0`) — Node.js установлен!

📸 *[скриншот: терминал с выводом версии node]*

## Шаг 3: Установи VS Code (опционально)

VS Code — удобный редактор кода. Не обязателен, но рекомендую.

### Установка

1. Перейди на **https://code.visualstudio.com**
2. Нажми **"Download"**
3. Установи:
   - **Mac:** Перетащи в папку Applications
   - **Windows:** Запусти установщик, Next → Next → Install

📸 *[скриншот: страница загрузки VS Code]*

### Зачем VS Code?

- Удобно видеть структуру папок
- Встроенный терминал
- Подсветка Markdown файлов

> Можно работать и без VS Code — только через терминал. Но с VS Code удобнее.

---

## Шаг 4: Установи Claude Code

Теперь установим сам Claude Code.

### 4.1 Открой терминал

- **Mac:** Terminal (Finder → Программы → Утилиты → Terminal)
- **Windows:** PowerShell (поиск → "PowerShell")
- **VS Code:** Нажми Ctrl+` (или Cmd+` на Mac) чтобы открыть встроенный терминал

📸 *[скриншот: терминал в VS Code]*

### 4.2 Установи Claude Code

Скопируй и вставь эту команду, нажми Enter:

```bash
npm install -g @anthropic-ai/claude-code
```

Дождись завершения (обычно 1-2 минуты).

📸 *[скриншот: процесс установки в терминале]*

### 4.3 Проверь установку

```bash
claude --version
```

Если видишь номер версии — Claude Code установлен!

---

## Шаг 5: Авторизуйся в Claude Code

При первом запуске нужно связать Claude Code с твоим аккаунтом.

### 5.1 Запусти Claude Code

В терминале введи:

```bash
claude
```

### 5.2 Авторизуйся

1. Claude Code покажет ссылку для авторизации
2. Автоматически откроется браузер (или скопируй ссылку вручную)
3. Войди в свой аккаунт Claude (тот, где оформлена подписка Max)
4. Нажми **"Allow"** (Разрешить)
5. Вернись в терминал — должно появиться подтверждение

📸 *[скриншот: страница авторизации в браузере]*

📸 *[скриншот: успешная авторизация в терминале]*

> 💡 **Проблема:** Если пишет "subscription required" — проверь, что подписка Max активна на https://claude.ai/settings

## Шаг 7: Подключи скелет к Claude Code

Теперь самое главное — подключаем скелет.

### Вариант A: Через терминал

1. Открой терминал

2. Перейди в папку со скелетом:

**Mac:**
```bash
cd ~/Projects/content-factory/content-factory-template
```

**Windows:**
```bash

```

> 💡 **Совет:** Можно перетащить папку в терминал — путь вставится автоматически

3. Запусти Claude Code:

```bash
claude
```

📸 *[скриншот: терминал с запущенным Claude Code]*

### Вариант B: Через VS Code (проще)

1. Открой VS Code

2. File → Open Folder (Ctrl+O / Cmd+O)

3. Выбери папку `content-factory-template`

📸 *[скриншот: VS Code с открытой папкой]*

4. Открой терминал в VS Code: View → Terminal (или Ctrl+`)

5. В терминале введи:

```bash
claude
```

📸 *[скриншот: Claude Code запущен в VS Code терминале]*

После запуска Claude Code ты увидишь приглашение для ввода.

```
привет
```

Claude должен ответить и предложить начать работу.

### Если Claude видит скелет, он скажет что-то вроде:

> "Вижу, что ты используешь Content Factory Skeleton..."

📸 *[скриншот: первое общение с Claude]*

## Шаг 9: Создай первый пост!

Теперь самое интересное — за 10 минут настроим профиль и создадим пост.

### 9.1 Запусти мини-интервью

Напиши Claude:

```
интервью мини
```

### 9.2 Ответь на 3 вопроса

Claude задаст три простых вопроса:

**Вопрос 1: Кто ты и чем занимаешься?**

Ответь коротко, 2-3 предложения. Пример:
> "Я психолог, работаю с тревожностью. 8 лет практики, провожу онлайн-консультации."

**Вопрос 2: Кому помогаешь?**

Опиши своего типичного клиента. Пример:
> "Женщины 30-45 лет, которые столкнулись с выгоранием на работе или после декрета."

**Вопрос 3: Что продаёшь?**

Твой продукт/услуга. Пример:
> "Консультации по 60 минут онлайн. Помогаю разобраться в причинах тревоги и составить план работы."

📸 *[скриншот: диалог мини-интервью]*

### 9.3 Сгенерируй посты

После интервью напиши:

```
threads 3
```

Claude создаст 3 поста для Threads на основе твоих ответов.

📸 *[скриншот: сгенерированные посты]*

### 9.4 Выбери лучший и опубликуй

- Прочитай все три поста
- Выбери тот, который больше нравится
- Скопируй текст
- Вставь в Threads
- Опубликуй!

🎉 **Поздравляю! Твой первый пост готов.**

## Частые проблемы

### "command not found: claude"

Node.js или Claude Code не установлены. Повтори шаги 2 и 4.

### "subscription required"

Подписка Max не активна. Проверь на https://claude.ai/settings

### "CLAUDE.md not found" / Claude не понимает команды

Ты запустил Claude Code не в той папке. Убедись, что:
1. Ты находишься внутри папки `content-factory-template`
2. В этой папке есть файл `CLAUDE.md`

### Терминал не понимает русские буквы

Попробуй VS Code — там обычно всё работает. Или используй английские команды:
- `interview mini` вместо `интервью мини`
- `threads 3` — работает одинаково

### Claude отвечает не по теме

Напиши `начать сначала` или просто закрой терминал и запусти `claude` заново.
