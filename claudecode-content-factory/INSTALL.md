# Установка Конструктора Фабрики Контента

> **Время:** 15-20 минут  
> **Уровень:** Не нужен опыт программирования  
> **Результат:** Готовая система для создания контента с ИИ

---

## Что тебе понадобится

Перед установкой убедись, что у тебя есть:

- [ ] **Claude Max** — подписка за $100/мес на [claude.ai](https://claude.ai)
- [ ] **Node.js 18+** — скачай с [nodejs.org](https://nodejs.org) (кнопка LTS)
- [ ] **Claude Code** — установим на шаге 1

---

## Шаг 1. Установи Claude Code

Открой **Терминал** (на Mac: Finder → Программы → Утилиты → Терминал) и введи:

```bash
npm install -g @anthropic-ai/claude-code
```

Подожди пока установится. Если видишь ошибки про `permission denied`, попробуй:

```bash
sudo npm install -g @anthropic-ai/claude-code
```

**Проверка:** Набери `claude --version` — должна показать версию.

---

## Шаг 2. Скачай конструктор

### Вариант A: Через GitHub Template (рекомендуется)

1. Открой конструктор: https://github.com/Mticool/content-factory
2. Нажми зелёную кнопку **"Use this template"** → **"Create a new repository"**
3. Назови как хочешь (например `my-content-factory`), нажми **Create**
4. Склонируй СВОЮ копию:

```bash
git clone https://github.com/ТВОЙ_ЮЗЕРНЕЙМ/my-content-factory.git
cd my-content-factory
```

### Вариант B: ZIP-архив

1. Открой https://github.com/Mticool/content-factory
2. Нажми зелёную кнопку **Code** → **Download ZIP**
3. Распакуй архив в удобное место
4. Переименуй папку в `content-factory` (без лишних суффиксов)

## Шаг 3. Разберись в структуре

После распаковки у тебя будет такая структура:

```
content-factory/
│
├── CLAUDE.md                 ← Главный файл (мозг системы)
├── VERSION                   ← Версия конструктора
├── CHANGELOG.md              ← Что нового в обновлениях
│
├── skills/           ← Навыки (методики контента)
│   ├── threads/              
│   ├── youtube/              
│   ├── carousel/             
│   ├── reels/                
│   ├── selling-meanings/     
│   └── ...                   
│
├── agents/                   ← Агенты (виртуальная команда)
│   ├── production/           
│   ├── analytics/            
│   └── strategy/             
│
├── projects/                 ← Проекты (ТВОИ данные)
│   ├── _template/            ← Шаблон для новых проектов
│   └── example-psychologist/ ← Пример готового проекта
│
├── docs/                     ← Документация
└── scripts/                  ← Вспомогательные скрипты
```

### 🔴 Что НЕ трогать

| Файл/папка | Почему нельзя |
|------------|---------------|
| `CLAUDE.md` (корневой) | Это «мозг» — без него система не работает |
| `skills/*` | Навыки обновляются автоматически при обновлении |
| `agents/*` | Агенты обновляются при обновлении |
| `projects/_template/` | Это шаблон — нужен для создания новых проектов |

### 🟢 Что будет только твоим

| Файл/папка | Для чего |
|------------|----------|
| `projects/{твой-проект}/brand/` | Твой голос и стиль |
| `projects/{твой-проект}/learning/` | Что сработало / не сработало |
| `projects/{твой-проект}/content/` | Черновики и готовый контент |
| `projects/{твой-проект}/strategy/` | Твоя стратегия и продукты |

## Шаг 4. Создай свой первый проект

### 4.1. Скопируй шаблон

В терминале, находясь в папке `content-factory`:

```bash
cp -r projects/_template projects/моё-имя
```

Замени `моё-имя` на название проекта **латиницей, без пробелов**. Например:

```bash
cp -r projects/_template projects/daria
cp -r projects/_template projects/dr-ivanov
cp -r projects/_template projects/coach-elena
```

### 4.2. Запусти Claude Code

```bash
claude
```

При первом запуске он попросит авторизоваться — следуй инструкциям на экране.

### 4.3. Напиши команду `начать`

Просто напиши в чат с Claude:

```
начать
```

Или: `/setup`

Claude проведёт мини-интервью (3 вопроса, ~10 минут):
1. **Кто ты?** — профессия, опыт, экспертиза
2. **Кому помогаешь?** — кто клиенты, с чем приходят
3. **Как общаешься?** — на «ты» или «Вы», стиль, запрещённые слова

После интервью Claude автоматически заполнит файлы в `brand/` и сгенерирует первые 3 поста.

## Шаг 5. Проверка, что всё работает

После настройки попробуй эти команды:

### Тест 1: Генерация постов
```
threads 3
```
**Ожидаемый результат:** 3 поста для Threads в твоём стиле

### Тест 2: YouTube-сценарий
```
youtube script про [твою тему]
```
**Ожидаемый результат:** Структурированный сценарий для видео

### Тест 3: Продающий контент
```
threads продающий пост про мой продукт
```
**Ожидаемый результат:** Пост с психологическими триггерами и CTA

### ✅ Всё работает, если:
- [ ] Посты звучат как ты (не как робот)
- [ ] Используются хуки и формулы (видно структуру)
- [ ] Есть твой тон — тёплый/профессиональный/как при настройке

### ❌ Не работает? Проверь:
- Есть ли файлы в `projects/{твой-проект}/brand/` — они должны быть заполнены
- Есть ли папка `skills/threads/` — в ней должны быть .md файлы
- Перезапусти Claude Code командой `claude` в терминале

## Дальнейшие шаги

### Углубить настройку голоса

```
интервью
```

Полное интервью (20-30 мин) — после него контент будет намного точнее попадать в твой стиль.

### Все доступные команды

| Команда | Что делает |
|---------|------------|
| `threads 5` | 5 постов для Threads |
| `youtube script [тема]` | Сценарий для YouTube |
| `carousel [тема]` | Карусель для Instagram |
| `reels [тема]` | Идея для Reels |
| `telegram [тема]` | Пост для Telegram-канала |
| `это сработало` | Запомнить удачный паттерн |
| `это не сработало` | Запомнить что не работает |
| `критика` | Оценка от 3 критиков |
| `инсайты` | Анализ что работает |

### Обучение системы

Чем больше работаешь — тем точнее контент:

1. **Генерируй** контент
2. **Публикуй** — смотри на отклик
3. **Давай обратную связь:** `это сработало` или `это не сработало`
4. **Повторяй** — система учится с каждым разом
