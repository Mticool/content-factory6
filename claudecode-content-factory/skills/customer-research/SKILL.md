# Unpacking


> Исследование даёт сырьё. Распаковка превращает его в готовые продающие тезисы.

## Что это

**Распаковка** — процесс перевода данных из исследования в:
- Обещания для продукта
- Тезисы о сервисе
- Отработку возражений
- Киллер-фичи
- Буллеты

Результат: карта смыслов из 8 столбиков, готовая для лендингов/прогревов/контента.

## Формула обещания

```
ТЫ + глагол (обещание результата) + как это будет реализовано
```

### Правило
Каждый тезис должен быть **ЖИРНО и СОЧНО** обоснован.
Чтобы хотелось купить даже от одной фразы.

### Пример

❌ **Плохо:**
```
"Ты внедришь услугу и станешь зарабатывать больше"
```

✅ **Хорошо:**
```
"Ты внедришь дополнительную услугу микроворонка
и минимум в 3 раза увеличишь свой средний чек.

Благодаря этому инструменту ты сможешь приводить 
клиенту прогретых клиентов, которые сразу готовы покупать."
```

## Методология кастдевов Егора Пырикова (запуск на 59.6 млн)

### Кого кастдевить (приоритет)

1. Те кто НЕ покупал (самые холодные — если научишься продавать им, тёплым будет легко)
2. Подписчики 1-3 месяца
3. Подписчики 6-12 месяцев
4. Подписчики >года
5. Те кто покупал (последние)

### Как проводить

- 2-3 человека на каждый сегмент
- Созвоны по 30-40 минут
- ВЫ + СЦЕНАРИСТ/КОНТЕНТЩИК (иногда делегируется)
- Конспектировать каждый созвон (помощник конспектирует)
- В конце созвона → закрывать на предоплату 5к (тут всплывают глубинные возражения)

### Что спрашивать (ключевое)

- Боли (шаблонные И глубинные)
- Потребности и желания
- Возражения (почему НЕ покупают)
- Триггеры (что мотивирует)
- "Ваш запрос на продукт" / "идеальный результат"

### Анализ аудитории (анкета)

- Норма заполнения: 7-15% от охватов
- Если 1% — звоночек: плохая лояльность аудитории
- Цель: узнать портрет, сегменты, проранжировать боли и желания

### Результат кастдевов → Карта смыслов

```
РАСПАКОВКА + АНАЛИЗ АУДИТОРИИ + КАСТДЕВЫ = КАРТА СМЫСЛОВ (20-30 штук)
```

Формат записи:
```
БОЛЬ: [конкретная формулировка из кастдева]
ЗАКРЫТИЕ: [как продукт/эксперт решает]
ДОКАЗАТЕЛЬСТВО: [кейс/цифра]
```

> ❗ Результаты кастдевов НЕ должны остаться в табличке. Выписать боль → накидать способы закрытия → внедрить в прогрев

### Правило обновления

> Кастдевы ПЕРЕД КАЖДЫМ запуском. Тренды меняются, рынок меняется, глаз замыливается. Нельзя использовать старые кастдевы.

### Кастдевы как продажи

- Если человек говорит "классный продукт" — пригласи на продукт, предложи за 5к забронировать
- Именно тут всплывают ГЛУБИННЫЕ возражения, которые потом закрываются в прогреве
- 1 созвон = выводы + попытка закрытия на предоплату
