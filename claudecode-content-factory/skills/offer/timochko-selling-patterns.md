## Философия: Почему Тимочко и Hormozi говорят одно и то же

Центральный тезис обоих подходов **один**:

```
Продажа = глубокое понимание реальности человека
        + предоставление лучшей альтернативы
```

**Тимочко:** «Экологичность продажи определяется продуктом, а не методом»
**Hormozi:** «Make people an offer so good they feel stupid saying no»

Оба говорят: если продукт реально решает проблему — продавать его **обязанность**, не наглость.

## I. КЭВ — Ключевой Этап Воронки

### Что такое КЭВ на глубинном уровне

КЭВ — это **не просто промежуточный шаг**. Это точка, в которой одновременно работают 4 из 15 когнитивных искажений:

| Искажение | Как КЭВ его активирует |
|-----------|----------------------|
| **Endowment Effect** | Клиент «попробовал» → начал ощущать владение результатом |
| **Reciprocity** | Получил бесплатную ценность → чувствует «должок» |
| **Consistency** | Сделал шаг (пришёл на вебинар) → мозг хочет быть последовательным и купить |
| **Sunk Cost** | Потратил 2 часа на вебинар → «не зря же я пришёл» |

### КЭВ через призму Value Equation Hormozi

```
              DREAM OUTCOME × LIKELIHOOD OF ACHIEVEMENT
VALUE = ────────────────────────────────────────────────────────
              TIME DELAY × EFFORT & SACRIFICE
```

**Что делает КЭВ с каждым элементом:**

| Элемент | Без КЭВ (прямые продажи) | С КЭВ |
|---------|--------------------------|-------|
| **Dream Outcome** | Абстрактное обещание | Клиент ВИДИТ результат на кейсах, ощущает его |
| **Likelihood** ↑↑ | «Может, сработает» | «Я видел доказательства, я верю» |
| **Time Delay** ↓ | «Когда-нибудь» | «Я уже понял первые вещи за 2 часа» |
| **Effort** ↓↓ | «Наверное, сложно» | «Мне показали путь из 5 шагов, это реально» |

**Вывод:** КЭВ — это **Likelihood-машина**. Он поднимает воспринимаемую вероятность успеха с 10% до 60-80%.

### Конструктор КЭВ: Как подобрать для любой ниши

**Шаг 1. Ответь на вопрос Тимочко:**
> «Чего должно быть БОЛЬШЕ, чтобы у вас было БОЛЬШЕ продаж?»

**Шаг 2. Заполни матрицу:**

| Параметр | Твой ответ |
|----------|-----------|
| Что клиент должен ОЩУТИТЬ перед покупкой? | ___ |
| Что даст ему «ага-момент» (я хочу это!)? | ___ |
| Что можно дать бесплатно/дёшево? | ___ |
| Как это логично ведёт к основному продукту? | ___ |

**Шаг 3. Выбери формат КЭВ:**

| Тип КЭВ | Лучше для | Пример |
|----------|-----------|--------|
| **Вебинар** (2-3 часа) | Массовые продажи, охваты 2000+ | Курс по запускам → вебинар «6 шагов к 1 млн» |
| **Консультация** (30-60 мин) | Дорогие продукты 100К+, до 2000 охватов | Наставничество → бесплатная стратегическая сессия |
| **Мини-курс** (3-5 видео) | Сложные продукты, длинный цикл принятия решения | Годовая программа → бесплатный 3-дневный курс |
| **Аудит/Разбор** (15-30 мин) | B2B, услуги | SMM-услуги → бесплатный аудит аккаунта |
| **Марафон** (2-5 дней) | Средний чек, нужно много доверия | Курс по питанию → 5-дневный марафон |
| **Пробный доступ** | SaaS, подписки, клубы | Закрытый клуб → 7 дней бесплатно |
| **Физическая проба** | Офлайн-услуги, товары | Фитнес-клуб → пробная тренировка |

### Математика КЭВ

**Без КЭВ:**
```
1000 подписчиков → Stories → Курс 50К
Конверсия: 1-3%
Продажи: 10-30 штук
Выручка: 500К – 1,5М
```

**С КЭВ (вебинар):**
```
1000 подписчиков → Регистрация на вебинар (20%) = 200
→ Доходимость (35%) = 70 человек
→ Досмотр до продажи (50%) = 35 человек
→ Конверсия в покупку (15-25%) = 5-9 штук
→ Выручка: 250К – 450К

НО:
+ Отдел продаж дожимает неоплаченные заявки (+30-50%)
+ Stories дожимают тех, кто не пришёл на вебинар (+10-30%)
+ Итого: 8-15 продаж = 400К – 750К
```

**При охватах 10К+:**
```
10 000 → 2 000 регистраций → 700 пришли → 350 досмотрели
→ 50-90 купили → 2,5М – 4,5М
+ дожим: 3,5М – 7М
```

## III. Конструктор продающего вебинара

### Философия вебинара

**Вебинар ≠ обучение. Вебинар = продажа.**

Это сборка **всего прогрева** за 2-3 часа для тех, кто:
- Не видел ваш контент
- Не читал ваши Stories
- Пришёл «посмотреть»

Вебинар — это КЭВ, который одновременно решает 5 задач:

```
1. ВДОХНОВИТЬ на тему → «Это реально возможно»
2. СОЗДАТЬ ДОВЕРИЕ → «Этот человек знает, что делает»
3. ПОКАЗАТЬ ПУТЬ → «Я вижу, как прийти к результату»
4. УБЕДИТЬ В ПРОДУКТЕ → «Этот курс = лучший инструмент»
5. ПРОДАТЬ → «Беру!»
```

### Пошаговый конструктор (блок за блоком)

---

#### БЛОК 0: ПОДГОТОВКА (за 2-7 дней до вебинара)

| Элемент | Что сделать | Зачем |
|---------|-------------|-------|
| **Лендинг регистрации** | Название + дата + 3 пункта «что узнаете» + бонус за регистрацию | Повышает конверсию в регистрацию vs прямая ссылка |
| **Напоминания** | За 1 день, за 2 часа, за 30 мин + ссылка | Доходимость: с 20% до 40%+ |
| **Бонус за присутствие** | «Всем, кто будет онлайн — гайд/чек-лист/урок» | Стимулирует прийти и остаться |
| **Предзапись** | Открыть форму «запишись первым со скидкой» | Сегментирует горячих, создаёт ажиотаж |
| **Тех. проверка** | Камера, звук, слайды, ссылки, оплата | Проблемы = потеря доверия |

---

#### БЛОК 1: INTRO (0-10 мин)

**Цель:** Захватить внимание, установить правила, создать ожидание.

```
📋 ШАБЛОН:

[Энергичное приветствие, НЕ «привет, меня зовут...»]

"Сегодня за 2 часа вы узнаете [большое обещание вебинара].
И в конце я покажу инструмент, который [конкретный результат]."

[Правила:]
"1. Пишите в чат — я читаю и отвечаю
 2. Останьтесь до конца — будет подарок [конкретный]
 3. Записывать не нужно — будет запись (но не на долго)"

[Самопрезентация — 2-3 предложения:]
"Меня зовут [имя]. За [N лет] я [главный результат].
Но об этом позже — сейчас главное: ваш результат."

[Программа вебинара — 3-5 пунктов:]
"Сегодня разберём:
 1. [Вдохновляющий пункт]
 2. [Практический пункт]
 3. [Секретный пункт]"
```

**Триггеры этого блока:** Любопытство, Consistency (сел → досидит), Reciprocity (обещан подарок)

---

#### БЛОК 2: ВДОХНОВЕНИЕ (10-60 мин)

**Цель:** Зажечь веру в тему. Показать, что возможности РЕАЛЬНЫ.

**Подблоки:**

**2.1. Актуальность темы (10-20 мин)**
```
— Почему эта тема важна СЕЙЧАС
— Тренды рынка, цифры, графики
— «Окно возможностей» (FOMO)
— Примеры людей, которые уже зарабатывают/живут так
```

**2.2. Возможности (20-35 мин)**
```
— Кейсы: 3-5 историй успеха (разные сегменты ЦА!)
— Цифры: доходы, результаты, метрики
— «Ещё 3 года назад это было невозможно, а сейчас...»
— Мировые примеры + российские
```

**2.3. Разрыв шаблона (35-45 мин)**
```
— «Но вот что большинство делает НЕПРАВИЛЬНО...»
— Развенчание 2-3 мифов с цифрами
— «Вот почему у 90% не получается»
— Переход: «И я хочу показать вам другой путь»
```

**Триггеры:** Bandwagon (все уже делают), FOMO (упустите), Authority (цифры и факты), Pattern Interrupt (мифы)

**Связь с Hormozi:** Этот блок прокачивает **Dream Outcome** — «я хочу так же!»

---

#### БЛОК 3: ДОВЕРИЕ (45-75 мин)

**Цель:** Установить экспертность. «Почему мне стоит слушать этого человека?»

**Подблоки:**

**3.1. Личная история (45-60 мин)**
```
— Откуда я пришёл (точка А — похожая на ЦА!)
— Трудности, ошибки, «тёмный период»
— Поворотный момент
— Текущие результаты (точка Б)
— «И я хочу провести вас этим путём быстрее»
```

**Правило похожести (Similarity):** История должна начинаться из точки, УЗНАВАЕМОЙ для аудитории. Не «я всегда был успешен», а «я тоже сидел без клиентов и не знал, что делать».

**3.2. Регалии и результаты (60-75 мин)**
```
— Цифры: выручка, количество клиентов/учеников
— Регалии: сертификаты, публикации, спикерство
— Результаты учеников (не только свои!)
— НЕ хвастаться — показывать через призму «поэтому я знаю, как»
```

**Триггеры:** Authority, Social Proof, Similarity, Reciprocity

**Связь с Hormozi:** Этот блок прокачивает **Likelihood of Achievement** — «этот человек знает путь, я смогу тоже»

---

#### БЛОК 4: ВИДЕНИЕ ПУТИ (75-110 мин)

**Цель:** Показать конкретные шаги от точки А к точке Б. Это ЯДРО вебинара.

**Структура:**

```
«Итак, вот путь, который проходят мои лучшие ученики:»

ШАГ 1: [Название] — «Сначала нужно...»
├── Что делать (коротко)
├── Пример/кейс
└── Типичная ошибка на этом шаге

ШАГ 2: [Название] — «Затем...»
├── Что делать (коротко)
├── Пример/кейс
└── «А вот тут большинство застревает...»

ШАГ 3: [Название] — «И ключевой момент...»
├── Самый важный инсайт
├── «Именно этот шаг отличает успешных от неуспешных»
└── Пример: «Маша на этом шаге сделала X и получила Y»

[Опционально: шаги 4-5]

«Видите, путь не такой сложный, как казалось?
Но есть нюансы, которые определяют результат.
И именно они — внутри программы.»
```

**Критически важно:** Давать **достаточно**, чтобы человек поверил в путь, но **не всё** — чтобы он понял: нужна система, наставник, программа.

**Тимочко:** «Вебинар — это не обучение. Это показ пути.»
**Hormozi:** «Give them the WHAT and the WHY, sell them the HOW.»

**Триггеры:** Framing (сложное выглядит простым), Confirmation Bias (подтверждает их мечту), Least Effort (путь понятен и выглядит реальным)

**Связь с Hormozi:** Снижает **Time Delay** (путь видно) и **Effort** (выглядит делабельно)

#### БЛОК 5: ПРОДУКТ (110-140 мин)

**Цель:** Презентовать продукт как **лучший инструмент** для прохождения пути.

**5.1. Переход от контента к продукту:**
```
«Я показал вам путь из [N] шагов.
Вы можете пойти сами — и это здорово.
Но если хотите пройти этот путь быстрее, безопаснее
и с моей поддержкой — я создал для вас инструмент.»
```

**5.2. Презентация программы:**
```
«Программа [название курса]:

Модуль 1: [Сочное название] → Результат: [конкретный]
Модуль 2: [Сочное название] → Результат: [конкретный]
...
Модуль N: [Сочное название] → Результат: [конкретный]

По итогу:
✅ [Конкретный результат 1]
✅ [Конкретный результат 2]
✅ [Конкретный результат 3]»
```

**Правило Тимочко:** Названия модулей — СОЧНЫЕ:
- ❌ «Введение» → ✅ «Быстрый старт»
- ❌ «Основы» → ✅ «Фундамент на миллион»
- ❌ «Продажи» → ✅ «Машина продаж 2.0»

**5.3. «По итогу» (Тимочко) = конкретные пощупаемые результаты:**
```
Не «научитесь продавать», а:
✅ «Сделаешь 4 сайта своими руками»
✅ «2 уникальных проекта в портфолио»
✅ «Возьмёшь первые 2 проекта по 5-15К»
```

**5.4. Сравнение с альтернативами:**
```
| Альтернатива | Что получите | Цена | Время |
|-------------|-------------|------|-------|
| Самостоятельно | Хаотичные знания | Бесплатно | 1-3 года |
| Другие курсы | Теория без практики | 20-50К | 3-6 мес |
| [Наш курс] | Система + практика + поддержка | [цена] | [срок] |
```

**Цель:** Закрыть сделку. Стек ценности → Цена → Усилители → CTA.

**6.1. Offer Stack (Стек ценности):**

Это ВИЗУАЛЬНАЯ презентация всего, что получает клиент. Каждый элемент показывается отдельно, с его «стоимостью».

```

✅ [Основной курс: N модулей, M уроков]

═══════════════════════════════
Общая ценность: [сумма]₽
Ваша цена сегодня: [гораздо меньше]₽
```

**Правило Hormozi:** Ценность бонусов в восприятии > цена основного продукта.

**6.2. Тарифы (3-5 штук):**

```
БАЗА                    СТАНДАРТ ⭐              VIP
[цена]                  [цена × 2-3]             [цена × 5-10]
─────────────────       ─────────────────         ─────────────────
5 модулей               8 модулей                 Все модули

Доступ 3 мес            Доступ 6 мес              Доступ навсегда
                        + Бонус-модули            + VIP-встреча
                                                  + Разбор проекта
```

**Правило:** Средний тариф приносит 60-70% выручки (он для этого и создан).

**6.3. Усилители (Enhancers по Hormozi):**

```
ГАРАНТИЯ: «14 дней — не понравится, вернём каждый рубль»

SCARCITY: «В группе 30 мест — потому что куратор работает
           с каждым лично. Осталось [N]»

URGENCY: «Цена [текущая] действует только сегодня до полуночи.
          Завтра +[сумма]»

БОНУС ЗА СКОРОСТЬ: «Первым 10 оплатившим — [ценный бонус]»
```

**6.4. CTA (призыв к действию):**
```
«Ссылка в чате / под видео / в описании.
Нажимаете → выбираете тариф → оплачиваете.
Через 5 минут получаете доступ.
Есть вопросы? Пишите — отвечу прямо сейчас.»
```

**6.5. Работа с возражениями в реальном времени (10-20 мин):**
```

— «Не хватает времени» → «30 мин в день, мобильный формат»
— «Не уверен, что получится» → Гарантия + кейс похожего человека
— «Подумаю» → «Конечно, подумайте. Только помните, что цена...»
```

## IV. CustDev: Фреймворк глубинного интервью

### Когда проводить CustDev

**ДЕЛАТЬ, когда:**
- Продюсер БОГАЧЕ ЦА → оторвался от реальности, не понимает ценность «дешёвого»
- Продюсер БЕДНЕЕ ЦА → не понимает, зачем платить 500К за курс
- Запуск дорогого продукта (100К+)
- Кардинальная смена ниши
- Первый запуск с новым экспертом

**НЕ делать, когда:**
- Меньше 2 недель до запуска
- Нет ассистента для расшифровки
- Примерно один уровень с ЦА (можно обойтись анкетой)

**Тимочко, цитируя Форда:** «Если бы я спросил людей, чего они хотят, они бы попросили более быструю лошадь» → Интервью показывает ПРОБЛЕМЫ, а не решения. Решения создаёте вы.

### 9 пунктов изучения аудитории

```
ПОВЕРХНОСТНЫЙ УРОВЕНЬ (опросы, Stories, анкеты):
├── 1. Общий срез (пол, возраст, регионы)
├── 2. Ключевые портреты (мамочки, студенты, бизнес)
├── 3. Платёжеспособность (месячный доход)
├── 4. Осведомлённость в сфере (новички/профи)
└── 5. Ключевые запросы (что горит)

ГЛУБИННЫЙ УРОВЕНЬ (только через интервью 1-на-1): ❤️
├── 6. Цели и хотелки — на что РЕАЛЬНО потратят деньги
├── 7. Преграды и боли — что конкретно не получается
├── 8. Критерии выбора — на что смотрят при покупке
└── 9. Конкуренты — с кем сравнивают, что уже пробовали
```

### 4 этапа CustDev

**Этап 1: Подготовка (1-2 дня)**

```
1. Определить цель исследования
   «Я хочу узнать: [конкретный вопрос]»

2. Составить скрипт (12-15 вопросов, 40-60 мин)

3. Найти 10-12 респондентов
   — Текущие подписчики (не покупавшие!)
   — Покупавшие похожие продукты у конкурентов
   — Люди из целевого сегмента

4. Назначить время + подготовить диктофон/запись
```

**Этап 2: Проведение интервью (1-2 недели)**

```
6 ЗОЛОТЫХ ПРАВИЛ (по Мире Таналиевой):

1. 🤝 Настроить ДОВЕРИЕ первым делом
   «Спасибо, что согласились. Нет правильных ответов.
   Мне важно ваше реальное мнение.»

2. 🚫 НЕ ПРОДАВАТЬ свой продукт!
   Цель — слушать, не убеждать.

3. ⏪ Спрашивать о ПРОШЛОМ и НАСТОЯЩЕМ, не о будущем
   ❌ «Купили бы вы курс за 50К?»
   ✅ «Сколько вы потратили на обучение за последний год?»

4. 🎯 НЕ принимать АБСТРАКТНЫЕ ответы
   ❌ «Было хорошо» → «Что конкретно было хорошо?»
   ❌ «Дорого» → «Сколько вы ожидали заплатить?»

5. 🙊 НЕ НАВОДИТЬ на ответы
   ❌ «Вам же понравился этот подход, да?»
   ✅ «Что вы думаете об этом подходе?»

6. 📝 ФИКСИРОВАТЬ инсайты прямо в процессе
   Дословные фразы = золото для продающих текстов.
```

**Этап 3: Анализ (2-3 дня)**

```
1. Транскрибировать все записи

2. Выделить повторяющиеся паттерны:
   — Какие боли называли 5+ из 10?
   — Какие слова/фразы повторяются?
   — Что удивило вас?

3. Составить матрицу находок:
   | Боль/Желание | Кто говорил | Цитата | Частота |
   |-------------|-------------|--------|---------|
   | [боль 1]    | Респ. 1,3,7 | «...»  | 3/10    |
   | [желание 1] | Респ. 2,4,8 | «...»  | 3/10    |

4. Ранжировать по частоте и интенсивности
```

**Этап 4: Генерация идей (1 день)**

```
Для каждой топ-3 боли:
├── Как это учесть в продукте?
├── Какими словами описать в продающих текстах?
└── Как показать, что наш продукт это решает?

Для каждого топ-3 желания:
├── Как вписать в Большое обещание?
├── Какой кейс показать?
└── Как визуализировать результат?

Для каждого критерия выбора:
├── Как показать, что мы соответствуем?
└── Как выделиться от конкурентов по этому критерию?
```

### Готовый скрипт CustDev (для агента)

**Используй этот скрипт как основу, адаптируй под нишу:**

```markdown
## Скрипт глубинного интервью: [Ниша]

### Разогрев (2-3 мин)
1. «Расскажите немного о себе — чем занимаетесь?»
2. «Как давно вы в [сфера]?»

### Блок 1: Текущая ситуация (10 мин)
3. «Как у вас сейчас обстоят дела с [ключевая тема]?»
4. «Что вас больше всего напрягает/не устраивает?»
5. «Можете вспомнить конкретную ситуацию, когда это было особенно остро?»
6. «Сколько это стоит вам — в деньгах, времени, нервах?»

### Блок 2: Прошлый опыт решения (10 мин)
7. «Что вы уже пробовали, чтобы решить эту проблему?»
8. «Какие курсы/продукты/услуги покупали по этой теме?»
9. «Что из этого помогло? Что нет? Почему?»
10. «Сколько вы потратили на это за последний год?»

### Блок 3: Идеальный результат (10 мин)
11. «Если бы вы могли щёлкнуть пальцами — что бы изменилось?»
12. «Как выглядит идеальный день после решения проблемы?»
13. «Что бы вы сделали с деньгами/временем, которые освободятся?»

### Блок 4: Критерии выбора (10 мин)
14. «Если бы вы выбирали курс/наставника — на что смотрите?»
15. «Что бы вас точно оттолкнуло?»
16. «Какой формат обучения для вас идеален?»

### Завершение (5 мин)
17. «Что я не спросил, но стоило бы?»
18. «Если бы вы могли получить 1 вещь — что бы это было?»

### Мои наблюдения (после интервью)
— Ключевые цитаты:
— Главный инсайт:
— Неожиданное:
— Что использовать в продажах:
```

### Кейс Porsche Cayenne (Тимочко)

**Как CustDev создал бестселлер:**
- Porsche изучил домохозяек (не автоэнтузиастов!)
- Узнали: нужно возить детей + продукты + безопасно + статусно
- Создали Cayenne: большой багажник + задние сидения + Porsche badge
- Результат: 20 000 продаж в первый год
- **Урок:** Продукт создаётся под РЕАЛЬНЫЕ потребности, не под предположения

### Связь CustDev с Value Equation Hormozi

```
CustDev →  Блок «Текущая ситуация»    → Dream Outcome (чего хотят НА САМОМ ДЕЛЕ)
CustDev →  Блок «Прошлый опыт»        → Effort (что уже не работало, что упростить)
CustDev →  Блок «Идеальный результат»  → Dream Outcome + Time Delay
CustDev →  Блок «Критерии выбора»      → Likelihood (что им нужно, чтобы поверить)
```

CustDev буквально **заполняет Value Equation данными из реальности**.

## V. Модели продаж: Когда что использовать

### 8 моделей + рекомендации

| Модель | Охваты | Цена продукта | Когда выбирать | КЭВ? |
|--------|--------|---------------|----------------|------|
| **Stories** | Любые | До 20К | Дешёвые продукты, tripwire | Нет |
| **Бесплатный вебинар** | 2000+ | 20К-200К | Лучший универсальный инструмент | **Да — это и есть КЭВ** |
| **Платный вебинар** (490-990₽) | 1000+ | 50К-500К | Отсеивает нецелевых, выше конверсия | Да |
| **Личные продажи** | До 2000 | Любая | Маленькая аудитория, дорогие продукты | Да (консультация) |
| **Предзапись + отдел продаж** | 3000+ | 30К-300К | Усилитель к любой модели | Нет (усилитель) |
| **Бесплатный марафон** | 2000+ | 30К-100К | Нужно больше доверия | Да (мини-КЭВ × 3) |
| **Платный марафон** | 1000+ | 100К+ | Премиум, высокая вовлечённость | Да |
| **Бесплатный курс** (3 видео) | 5000+ | 50К-500К | Сложный продукт, длинный цикл решения | Да |

### Чередование моделей

**Модели «выгорают»** — аудитория привыкает к формату. Чередуйте:

```
Запуск 1: Бесплатный вебинар     (классика)
Запуск 2: Платный марафон         (новый формат)
Запуск 3: Бесплатный курс        (глубина)
Запуск 4: Stories + личные        (отдых от «событий»)
Запуск 5: Бесплатный вебинар     (вернулись, но обновили)
```

### Комбинирование моделей

**Мощная комбо:** Вебинар + Предзапись + Отдел продаж + Stories-дожим

```
За 2 недели: прогрев в Stories
За 3 дня: открыть предзапись (ажиотаж)
В день X: вебинар (основные продажи)
1-3 дня после: отдел продаж обзванивает заявки
1-10 дней после: Stories-дожим

Результат: максимальная выручка из одного запуска
```

## VI. Высокие цены и продуктовая линейка

### Добродетельный цикл цены (Тимочко × Hormozi)

```
     ┌──── Высокая цена ────┐
     │                       │
     ↓                       │
Лучший продукт         Больше кейсов
и прогрев              и рекомендаций
     │                       ↑
     ↓                       │
Клиенты более           Лучшие результаты
вовлечены               клиентов
     │                       ↑
     └───────────────────────┘
```

**Данные Тимочко:**
| Продукт | Цена | Покупки | Выручка |
|---------|------|---------|---------|
| Закулисье | 7 900₽ | 1 700 | 13,4М |
| Формула запуска | 100 000₽ | 1 100 | 110М |

- Разница в чеке: **13x**
- Разница в продажах: всего **30-40%** (не в 13 раз!)
- Разница в выручке: **~10x**

**Вывод:** «Сделать 100 продаж по 50К гораздо легче, чем 5000 продаж по 1К» (Тимочко)

### Продуктовая линейка = Value Ladder

| Уровень | Цена | Цель | Частота | Пример Тимочко |
|---------|------|------|---------|----------------|
| **TRIPWIRE** | 3-15К | Вовлечь, прогреть, показать качество | Каждые 1-2 мес | Закулисье 7 900₽ |
| **ОСНОВНОЙ** | 50-200К | Главная выручка | 2-3 раза/год | Формула запуска 120 000₽ |
| **ПРЕМИУМ** | 500К-2М | Удвоить выручку, работа с лучшими | 1-2 раза/год | Мастер-группа 1 500 000₽ |

**Результат Тимочко:** +97 млн рублей за год только от расширения линейки.

### Принципы линейки

```
1. ❌ Нельзя основной продукт продавать каждый месяц
   → Сломается логика премиальности и запуска

2. ✅ Между основными запусками — tripwire и премиум
   → Аудитория «тёплая», деньги идут

3. ✅ Каждый продукт для СВОЕГО сегмента
   → Tripwire: «попробовать», Основной: «результат», Премиум: «трансформация»

4. ✅ Логичная эволюция: простое → сложное
   → Клиент растёт вместе с линейкой
```

## VII. Большое обещание: Конструктор

### Формула (Тимочко + Hormozi)

```
БОЛЬШОЕ ОБЕЩАНИЕ = [Конкретный результат] + [Срок] + [Условие «без»]
```

**Через призму Value Equation:**
- `[Конкретный результат]` → Dream Outcome ↑
- `[Срок]` → Time Delay ↓
- `[Условие «без»]` → Effort ↓

### Примеры

| Обещание | Dream Outcome | Time | Effort |
|----------|--------------|------|--------|
| «5 сайтов с нуля, доход от 70К» | 70К/мес + навык | — | с нуля (можно!) |
| «Стройная талия за 2 месяца без диет» | Стройность | 2 мес | без диет |
| «Первый клиент за 21 день без опыта» | Первый клиент | 21 день | без опыта |
| «30% годовых в долларах безопасно» | 30% доход | 1 год | безопасно |

### Антипаттерны

```
❌ «Измените свою жизнь» → Абстрактно, не измеримо
❌ «Научитесь зарабатывать» → Нет конкретики, нет срока
❌ «Станьте успешным экспертом» → Кто решает, что успешный?
```

### Правило безопасного роста

**Тимочко:** Большие скачки в доходах опасны для психики. Градация:
```
10К → 30К → 100К → 300К → 700К → 1М
```

**В обещаниях:** Не обещать прыжок с 50К на 5М — не поверят. Обещай следующую ступеньку.

## VIII. Создание программы курса: «Продукт на 1 млн$»

### 8 шагов (Тимочко)

```
СНАЧАЛА — КОНЦЕПТ (как продавать):
1. Исследование аудитории → Знаю, КОМУ продавать
2. Большое обещание → Знаю, ЧТО обещать
3. Программа курса → Знаю, какой ПУТЬ показать
4. Результаты «по итогу» → Знаю, что КОНКРЕТНО получит клиент
5. Форматы → Знаю, КАК проводить
6. Обратная связь → Знаю, какой СЕРВИС дать
7. Тарифы и цены → Знаю, СКОЛЬКО стоит
8. Вау-фишки → Знаю, чем УДИВИТЬ

ПОТОМ — МЕТОДОЛОГИЯ (как создавать):
→ Контент, уроки, материалы
```

**Философия:** «Можно провалиться с хорошим продуктом, но плохой упаковкой. Нельзя провалиться с хорошей упаковкой — плохой продукт доделаешь.»

### Конкурентный анализ: Конструктор

**Что изучать у конкурентов:**
1. Цены и тарифы → Скорректировать свои
2. Программа → Включить всё, что у них + своё уникальное
3. Схема продаж → Понять их воронку
4. Вебинар → **Посмотреть полностью!** (= минимум 20 идей)
5. Сайт → Что есть у них, чего нет у нас

**Формула Тимочко:** Лучший запуск = знать конкурентов + взять их лучшие решения + не повторять ошибки

## IX. Reels-воронки и контент-стратегия

### Архитектура Reels-воронки

```
REELS (массовый охват, бесплатный трафик)
  │
  ├── Кодовое слово в комментариях / описании
  │
  ↓
Instagram-бот (автоматический ответ)
  │
  ├── «Спасибо! Лови подарок: [лид-магнит]»
  │
  ↓
Перегон в Telegram
  │
  ├── Лид-магнит доставлен
  ├── Автоматическая цепочка прогрева
  │
  ↓
Продающая последовательность → Покупка
```

**Почему это мощно:** Reels показываются ПОСТОЯННО → трафик идёт 24/7 → автоматизация масштабирует.

### 5 типов контента (по Тимочко)

| Тип | Для кого | Задача | Конверсия в подписку |
|-----|----------|--------|---------------------|
| **Экспертный** | Новая аудитория (контент-аут) | Привлечь целевых | ★★★★★ |
| **Развлекательный** | Подписчики (контент-ин) | Удержать | ★★☆☆☆ |
| **Мотивирующий** | Подписчики (контент-ин) | Вдохновить | ★★★☆☆ |
| **Личный** | Подписчики (контент-ин) | Создать связь | ★★★☆☆ |
| **Продающий** | Подписчики (контент-ин) | Конвертировать | ★☆☆☆☆ (прямо) |

**Правило:** На маленькой аудитории — 100% экспертный контент.

### RPG-подход к персонажу (Тимочко)

```
Персонаж в соцсетях = герой в игре

Характеристики для «прокачки»:
├── 🎨 Визуал (свет, картинка, одежда)
├── 🎤 Голос (дикция, энергия, тембр)
├── 🧠 Смыслы (глубина, экспертиза)
├── 🎬 Подача (ритм, монтаж, формат)
├── 💄 Образ (стиль, харизма)
└── 📱 Техника (камера, звук, свет)

Каждое улучшение = +N очков к привлечению
```

### Производственный процесс

```
1 съёмочный день → 15-20 роликов → контент на 1-2 месяца

Процесс:
День 1: Генерация 30 идей
День 2: Написание сценариев
День 3: Съёмка (несколько образов)
Дни 4-5: Монтаж
→ Публикация: раз в 2-3 дня (качество > частота)
```

## X. Психология продаж: Объединённые паттерны

### Развитие по ступенькам (Тимочко)

```
Видна только СЛЕДУЮЩАЯ ступенька:

Уровень 5: Поток клиентов ← (видна с уровня 4)
Уровень 4: Твёрдый кейс ← (видна с уровня 3)
Уровень 3: Пробные проекты ← (видна с уровня 2)
Уровень 2: Тренировка на себе ← (видна с уровня 1)
Уровень 1: Изучение навыка ← ТЫ ЗДЕСЬ
```

**В продажах:** Не показывай всю лестницу — пугает. Покажи СЛЕДУЮЩИЙ шаг.
**В контенте:** Не «станешь миллионером через год», а «через месяц получишь первого клиента».

### Продажа энергией (без опыта)

```
Горящие глаза + конкретные решения = закрытие без портфолио

Компоненты:
1. ЭНТУЗИАЗМ — искренняя вера в результат
2. КОНКРЕТИКА — не «помогу с продвижением», а «сделаю 3 Reels + аудит воронки»
3. ПОГРУЖЕНИЕ — «я изучил ваш аккаунт, вижу потенциал в X»
```

### Клиент не ищет лучшего — ищет подходящего

```
У клиента НЕТ сравнительной таблицы всех продюсеров/экспертов.
Ему нужен ПОДХОДЯЩИЙ, а не ЛУЧШИЙ.

В позиционировании:
❌ «Мы лучшие на рынке»
✅ «Идеально для мам в декрете, которые хотят зарабатывать из дома»

В продажах:
❌ Доказывать, что лучше конкурентов
✅ Показать, что ПОНИМАЕШЬ клиента и его ситуацию
```

### Экологичность через Value Equation

```
Если VALUE = (Dream Outcome × Likelihood) / (Time × Effort)

И ваш продукт РЕАЛЬНО:
- Даёт конкретный результат (Dream Outcome)
- С высокой вероятностью (Likelihood)
- Быстро (Time)
- Без чрезмерных усилий (Effort)

Тогда НЕ продать его = НЕЭКОЛОГИЧНО.
Вы лишаете человека лучшей альтернативы.
```

## XI. Чек-листы и шаблоны

### Чек-лист запуска

```
ПОДГОТОВКА (за 2-3 месяца):
□ CustDev / анкетирование проведено
□ Конкурентный анализ (посмотрен хотя бы 1 вебинар!)
□ Большое обещание сформулировано (результат + срок + без)
□ Программа курса с сочными названиями
□ «По итогу» — конкретные результаты
□ КЭВ определён и формат выбран
□ Тарифы и цены (3-5 штук)
□ Модель продаж выбрана
□ Продуктовая линейка продумана

КОНТЕНТ (за 1 месяц):
□ 30-дневный контент-план
□ Прогрев в Stories/постах
□ Reels-воронка настроена (если есть)
□ Email/Telegram-последовательности
□ Лендинг готов

ТЕХНИКА:
□ Платёжная система
□ CRM / таблица учёта
□ Отдел продаж (или план личных продаж)
□ Система напоминаний (вебинар)

ЗАПУСК И ПОСЛЕ:
□ Вебинар проведён (или другой КЭВ)
□ Дожим: отдел продаж + Stories (1-10 дней)
□ Метрики собраны
□ Ошибки зафиксированы для следующего запуска
□ Кейсы учеников собраны
```

## Ключевые цитаты-смыслы

> **«Чего должно быть БОЛЬШЕ, чтобы у вас было БОЛЬШЕ продаж?»** — вопрос для определения КЭВ

> **«Сначала концепт, потом методология»** — упаковка важнее начинки (на старте)

> **«Выстроить поток из сотен клиентов легче, чем искать одного»** — системный подход

> **«Сделать 100 продаж по 50К гораздо легче, чем 5000 продаж по 1К»** — высокие цены

> **«Устроиться менеджером по продажам к самому себе»** — поиск как работа

> **«Клиент не ищет лучшего — ищет подходящего»** — позиционирование

> **«Ребёнку для формирования нужно 9 месяцев. Мы можем поторопить, но ничего хорошего не выйдет»** — терпение

> **«Возможности везде. Серьёзно поставь задачу — точно найдёшь»** — mindset

> **«Решение создать новые продукты дало мне 97 миллионов дополнительных денег за год»** — продуктовая линейка
