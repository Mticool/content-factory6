# Ценности и принципы

## Зачем

Ценности — это **фильтр решений**. Когда непонятно что делать — смотришь на ценности.

В контенте ценности:
- Притягивают единомышленников
- Отсекают "не своих"
- Создают предсказуемость (люди знают, чего ждать)

## Как найти свои ценности

### Метод "Красная линия"
За что готов потерять клиента/деньги?
→ Это твоя ценность.

### Метод "Бесит"
Что бесит в нише?
→ Переверни = ценность.
- Бесит: "Все врут про результаты" → Ценность: честность
- Бесит: "Продают впустую" → Ценность: результат

### Метод "Гордость"
Чем гордишься в своей работе?
→ Это ценность.

## Формат записи

```
Ценность: {название}
Значит: {что конкретно делаю/не делаю}
Пример: {реальный случай}
```

## Как использовать в контенте

- Манифест-пост: "Я никогда не буду..."
- Провокация: "Все в нише делают X, а я считаю это вредным"
- История: "Однажды я потерял клиента, потому что..."
- Фильтр: "Если вы верите в X — нам не по пути"
