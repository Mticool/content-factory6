# Reels/Shorts/TikTok — Система вирусного контента


Полный цикл от идеи до продаж. Не просто "как снять рилс", а как построить машину продаж через вертикальный контент.

**Философия:** Reels — это НЕ контент. Reels — это верхушка автоматизированной воронки продаж.

## Быстрый старт

Создание вирусного ролика с продающим опенингом:

```bash
# Сценарий с ВИСП-опенингом
"reels script [ИДЕЯ]" 

# 10 вариантов опенинга
"reels opening [ТЕМА]"

# Проверка по формуле ВИСП
"check VHF: [текст опенинга]"
```

**Формула успеха:** Первые 3 секунды = 80% успеха ролика

## Параметры/Настройки

| Метрика | Нижняя | Норма | Верхняя |
|---------|--------|-------|---------|
| Retention на всём ролике | 50% | 65% | 80%+ |
| Retention 4+ сек | 60% | 70% | 80%+ |
| Конверсия в подписку | 0.05% | 0.17% | 0.6%+ |
| Конверсия в лайки | — | — | 2%+ |

**Правило:** 2+ выше верхней = успех. 1 ниже нижней = провал.

## Workflow

### 1. Тест "200 миллионов"
- Понятно КАЖДОМУ?
- Выгода универсальна?
- Привязана к боли/страху/желанию ЦА?

### 2. Выбор шаблона по цели

| Цель | Лучшие шаблоны |
|------|---------------|
| **Привлечение ЦА** | Инструкция, Секрет, Страх |
| **Максимальный охват** | Лайфхаки, Новости, Тренды |
| **Вовлечение** | Вызов, Топ |
| **Продажа** | Инструкция + CTA, Секрет + CTA |

### 3. Опенинг по формуле ВИСП

| Элемент | Проверочный вопрос |
|---------|-------------------|
| **В**ыгода | Это желание 5-летнего? |
| **И**нтрига | Ответ заменён на "это"? |
| **С**рочность | Есть повелительное наклонение? |
| **П**ричастность | Зритель узнаёт себя? |

**Минимум 2-3 элемента в каждом опенинге**

### 4. Структура сценария
```
Проблема → Актуализация → (Самопрезентация) → Решение → Вывод → CTA
```

### 5. Продающий CTA
- Подписка / Комментарий / Кодовое слово
- CTA по ВИСП, не "переходи по ссылке"

## Частые ошибки

❌ **Ответ сразу в опенинге** — главный убийца удержания  
❌ **Несколько тем в одном ролике** — 1 видео = 1 проблема  
❌ **"Топ-5 лучших"** — привет из 2000-х  
❌ **Развлекательный контент без экспертности** — нецелевая аудитория  
❌ **CTA "переходи по ссылке"** — слишком прямолинейно  
❌ **Игнорирование первых 3 секунд** — 80% успеха решается здесь  
❌ **Однообразная картинка** — нужны субтитры и перебивки

### Воронка (экспресс-формула)
```
REELS → Кодовое слово → Instagram бот → Telegram → Лид-магнит → КЭВ → Продажа
10K      3% (300)        80% (240)       50% (120)   25% (30)    35% = 10-15 продаж

При среднем чеке 100K₽ = 1-1.5 млн₽
```

### Диагностика проблем

| Симптом | Причина | Решение |
|---------|---------|---------|
| Мало просмотров | Слабые первые 3 сек | ВИСП-опенинг |
| Подписок нет | Нет CTA | "Здесь всё о..." |
| Продаж нет | Нет воронки | Кодовое слово → бот |
| Низкий retention | Ответ сразу | "Печёночный торт" |

### Правила устной речи в скриптах

**Скрипт — это НЕ статья.** Это устная речь. Проверяй:

1. **Прочитай вслух.** Если язык спотыкается, предложение слишком длинное, или звучит как доклад — переписывай.
2. **Добавь паузы.** Многоточие, обрыв мысли, "ну типа", "короче" — это нормально для устной речи.
3. **Не пиши как для текста.** "Осуществление данного процесса" → "Делаешь вот так". Устная речь проще и грязнее.
4. **Варьируй ритм.** Короткая фраза. Потом длинная с пояснением и деталями которые раскрывают мысль. Потом снова короткая.
5. **Тест "голосовое сообщение".** Если это не звучит как голосовое другу — это не скрипт для рилс.

### Ключевые техники
- **"Печёночный торт"** — ответ в самом конце
- **"Дед Мороз"** — выгода как желание 5-летнего
- **"200 миллионов"** — понятно каждому человеку на планете
- **RPG-подход** — каждое улучшение даёт +N к привлечению

## Reels при запуске (из разбора на 59.6 млн)

### Когда публиковать

Рилсы во время конверсионных этапов (анкета, регистрация на веб) = **+20% к заполнениям**

### Связка

```
Пост + Сторис + Рилс = связывать логически
```

Дублировать самые концентрированные мысли и призывы из прогрева в рилсы.

### Формула рилса при запуске

```
Боль/потребность/возражение → Закрытие кейсом/отсылкой к продукту
```

### Правила

- Несколько рилсов в неделю во время прогрева
- Особенно важно при низких охватах в сторис (посты/рилсы набирают больше охватов)
- Пользы давать меньше, чем давления (боли, триггеры, ажиотаж)
