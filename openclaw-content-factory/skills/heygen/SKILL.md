# HeyGen AI-Аватары


Полный цикл создания AI-аватаров: от написания скрипта до оживления фотографий с правильной озвучкой.

**Применение:** Персонализированные видео, обучающий контент, презентации с виртуальными спикерами.

## Быстрый старт

Подготовка текста для AI-озвучки:

```bash
# Адаптация текста под HeyGen
"Адаптируй этот текст для HeyGen: [ТЕКСТ]"

# Создание motion промпта для фото
"Создай motion промпт для оживления этого фото"

# Полный скрипт с правилами HeyGen
"Напиши скрипт для HeyGen-аватара про [ТЕМА]"
```

## Параметры/Настройки

| Параметр | Ограничение | Описание |
|----------|-------------|----------|
| Длина предложения | Макс 20 слов | Для естественной озвучки |
| Длительность видео | До 10 минут | Лимит HeyGen |
| Motion продолжительность | 5-10 секунд | На один промпт |
| Формат чисел | Словами | "500" → "five hundred" |

## Примеры использования

## Workflow

### 1. Подготовка текста (heygen-rules.md)
- Замена запрещённых символов
- Перевод чисел в словесный формат
- Разворачивание аббревиатур
- Ограничение длины предложений

### 2. Создание аватара (Photo Avatar)
- Подготовка качественного фото
- Генерация через Nano Banana если нужно
- Загрузка в HeyGen

### 3. Motion промпты (heygen-motion.md)
- Выбор подходящего движка (Kling рекомендован)
- Создание промпта для оживления
- Учёт ограничений времени

### 4. Склейка и финализация
- Разбивка длинных скриптов
- Создание переходов между сегментами
- Финальная обработка

## Частые ошибки

❌ **Запрещённые символы в тексте** — " " ... — – * & % / ( ) [ ]  
❌ **Числа цифрами** — писать словами "five hundred"  
❌ **Длинные предложения** — максимум 20 слов  
❌ **Неразвёрнутые аббревиатуры** — "BPH" → "B P H"  
❌ **Гипофора в тексте** — риторический вопрос + ответ сразу  
❌ **Плохое качество исходного фото** — нужен чёткий портрет  
❌ **Motion промпт больше 10 секунд** — разбивать на части

### Правила текста для озвучки

**Запрещённые символы:**
```
" " (кавычки)
... (многоточие)  
— – (тире)
* & % / ( ) [ ] (спецсимволы)
```

**Числа словами:**
- "500 долларов" → "five hundred dollars"
- "2024 год" → "twenty twenty four"
- "15%" → "fifteen percent"

**Аббревиатуры:**
- "BPH" → "B P H" (по буквам)
- "AI" → "A I" 
- "CEO" → "C E O"

**Длина предложений:**
- Максимум 20 слов
- Простые конструкции
- Избегать сложноподчинённых

### Motion движки

| Движок | Качество | Скорость | Промпты |
|--------|----------|----------|---------|
| **Kling** | Высокое | Средняя | Хорошо понимает |
| Runway | Среднее | Быстрая | Простые |
| Pika | Среднее | Быстрая | Ограничены |

**Рекомендация:** Kling для говорящих голов и презентаций

### Типовые motion промпты

**Деловое общение:**
```
Professional presenter gesturing while speaking, subtle head movements, confident body language, clean background
```

**Обучающий контент:**
```
Teacher explaining concept with hand gestures, pointing motions, engaging facial expressions, educational setting
```

**Дружественная беседа:**
```
Friendly person talking directly to camera, natural smile, relaxed gestures, conversational tone
```

### Склейка длинных видео

Для контента больше 10 минут:

1. **Разбивка скрипта** — сегменты по 8-9 минут
2. **Переходы** — логические связки между частями
3. **Consistency** — одинаковая одежда и фон
4. **Склейка** — smooth переходы в монтаже
