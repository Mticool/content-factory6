# Telegram Контент

Система создания контента для Telegram-канала: от ежедневных постов до запусковых воронок.

## Быстрый старт

```bash
# Пост для канала
"Напиши telegram пост на тему: [ТЕМА]"

# Контент-план
"Контент-план для telegram на неделю"

# Прогревочная серия
"Прогрев к запуску [ПРОДУКТ] на 7 дней"
```

## Принцип: Threads → Telegram

Threads = охваты и новая аудитория.
Telegram = глубина, доверие, продажи.

В Telegram аудитория уже тёплая. Здесь можно:
- Длинные посты (без лимита символов)
- Прямые ссылки на продукты
- Серии постов (прогревы)
- Голосовые и видеокружки

## Форматы постов

| Формат | Когда использовать | Длина |
|--------|-------------------|-------|
| Инсайт | Ежедневный контент | 500-1500 символов |
| Кейс | 2-3 раза в неделю | 1000-2500 символов |
| Закулисье | 1-2 раза в неделю | 300-800 символов |
| Продающий | По плану запуска | 1000-2000 символов |
| Подборка | 1 раз в неделю | Список из 5-10 пунктов |
| Голосовое | 2-3 раза в неделю | 2-5 минут |

## Структура поста

### Инсайт-пост
1. Хук (первые 2 строки видны в превью)
2. Контекст/история
3. Инсайт/вывод
4. Вопрос или CTA

### Продающий пост
1. Боль или желание
2. Мост (как решается)
3. Оффер (что получает)
4. Социальное доказательство
5. CTA с ссылкой

## Воронка прогрева (7 дней)

| День | Тип | Задача |
|------|-----|--------|
| 1 | Проблема | Показать что боль реальна |
| 2 | История | Личный опыт с этой болью |
| 3 | Инсайт | Неочевидное решение |
| 4 | Кейс | Результат клиента |
| 5 | Метод | Как это работает |
| 6 | Возражения | Ответы на сомнения |
| 7 | Оффер | Продажа с дедлайном |

## Правила

- Первые 2 строки = хук (видны без раскрытия)
- Эмодзи: умеренно, 2-4 на пост
- Форматирование: **жирный** для акцентов
- Ссылки: прямые, не сокращённые
- Время постинга: тестировать 9:00, 12:00, 18:00, 21:00

## Частые ошибки

❌ Копипаст из Threads — в Telegram другой формат и глубина
❌ Только продающие посты — нужен баланс 80/20
❌ Без форматирования — стена текста не читается
❌ Посты без CTA — каждый пост должен куда-то вести
❌ Нерегулярность — минимум 1 пост в день

## ТГ прогрев при запуске (из разбора на 59.6 млн)

### Параллельно с Инстой

ТГ = место где сильно расширяется глубина прогрева и касания с аудиторией. Даёт +10-50% выручки.

### Правила дублирования

- **50% смыслов** из Инсты, но в ДРУГОМ формате и с другого захода
- Менять формат каждые 1-2 поста
- Разные заходы к одним и тем же смыслам
- Повторять — но не больше 50%

### Структура закрепа (эталон)

```
1. Заголовок (цепляющий)
2. Самопрезентация с позиционированием + личные кейсы
3. Чем отличаюсь от конкурентов
4. О чём канал, чем полезен
5. Навигация по лучшим постам:
   - Первая строка → перевод в Инст на лид-магнит
   - Далее посты и эфиры под боли ЦА
6. Упоминание продукта + перевод на кейсы в Инст
7. Призыв подписаться на Инст + лид-магнит
```

### Форматы для прогрева

| Формат | Частота | Эффект |
|--------|---------|--------|
| Экспертные посты на боли | Каждый день | Основа прогрева |
| Голосовые подкасты | 10+ за прогрев | Высокая лояльность |
| Разборы/эфиры | 1/нед | 15-20% выручки |
| Кейсы (развёрнутые) | 2-3/нед | Социальное доказательство |
| Лайф/инфоповоды | По ситуации | Поддержка охватов |
| Длинный подкаст про продукт | 1 шт | Killer feature запуска |

### Длинный подкаст про продукт (must-have)

Голосовое/аудио где подробно рассказать:
- Для кого продукт
- Какой результат
- Как окупится
- Анонс анкеты
- Фишки программы
- Кто из топов уже идёт

> "Нужно чтобы люди просто прослушали и влюбились в продукт"

### Эфиры: усиление

- На каждом эфире призыв писать в Директ / заполнить анкету
- Прогревать к эфиру в Инст И в ТГ
- 2-3 анонса перед эфиром
- Триггерные таймкоды к записи (+20-40% просмотров)
- Форматы: контент + разборы, только разборы, разбор кейсов с обучения

### Что НЕ сделал Егор (можно внедрить)

- ❌ Бот в закрепе для лид-магнита + сбор базы для рассылки
- ❌ Спам-рассылка в день анкеты
- ❌ Канал с отзывами и кейсами
- ❌ Вторая часть эфира с разбором кейсов
