## Шаблоны постов (30+)

### Категория: Hook-first (Зацепка)

#### 1. Провокационный вопрос
```
[Провокационный вопрос, который заставит остановиться]

[2-3 предложения, раскрывающих проблему]

[Ваш взгляд на решение]

[CTA или вопрос к аудитории]
```
**Пример:** "Почему ваш контент никто не читает? Потому что вы пишете о себе, а не о них..."

#### 2. Контринтуитивное утверждение
```
[Утверждение, которое противоречит общепринятому мнению]

[Объяснение, почему это так]

[Доказательство или пример]

[Вывод]
```
**Пример:** "Чем больше вы работаете, тем меньше зарабатываете. Звучит странно? Объясню..."

#### 3. Цифра-шок
```
[Шокирующая статистика или цифра]

[Что это значит для читателя]

[Как избежать/получить]

[CTA]
```
**Пример:** "90% экспертов продают себя в 10 раз дешевле, чем могут. Вот почему..."

---

### Категория: Story (Истории)

#### 4. Личный провал
```
[Ситуация: когда и где это было]

[Проблема: что пошло не так]

[Последствия: что потеряли]

[Урок: что поняли]

[Совет: что делать вам]
```
**Пример:** "В 2019 году я потерял 2 млн на запуске. Вот какую ошибку я допустил..."

#### 5. Трансформация клиента
```
[Имя и контекст клиента]

[Ситуация "до"]

[Что сделали/изменили]

[Результат "после" с цифрами]

[Ключевой фактор успеха]
```
**Пример:** "Марина пришла с доходом 30К. Через 3 месяца — 150К. Что изменилось?..."

#### 6. До/После автора
```
[Я раньше думал/делал...]

[Потом понял/научился...]

[Теперь...]

[Главный вывод]
```
**Пример:** "Раньше я думал, что нужно работать больше. Теперь работаю 4 часа и зарабатываю в 3 раза больше. Вот что изменилось..."

---

### Категория: Lesson (Уроки)

#### 7. Одна ошибка
```
[Ошибка, которую делают многие]

[Почему это ошибка]

[К чему приводит]

[Как правильно]
```
**Пример:** "Главная ошибка при запуске курса — начинать с продукта, а не с аудитории..."

#### 8. Пошаговая инструкция
```
[Проблема, которую решаем]

Шаг 1: [действие]
Шаг 2: [действие]
Шаг 3: [действие]

[Результат, если следовать]

[CTA]
```
**Пример:** "Как повысить цены в 2 раза без потери клиентов. 3 шага:..."

#### 9. Список ошибок/советов
```
[Тема + количество пунктов]

1. [Пункт + краткое пояснение]
2. [Пункт + краткое пояснение]
3. [Пункт + краткое пояснение]
...

[Главный вывод]
```
**Пример:** "5 признаков того, что ваш контент не работает:..."

### Категория: Controversy (Провокация)

#### 10. Непопулярное мнение
```
Непопулярное мнение: [утверждение]

[Почему так думаете]

[Доказательство]

А вы как считаете?
```
**Пример:** "Непопулярное мнение: бесплатные вебинары убивают продажи..."

#### 11. Разрушение мифа
```
Миф: [распространённое заблуждение]

Реальность: [как на самом деле]

[Доказательство]

[Что делать вместо]
```
**Пример:** "Миф: нужно много подписчиков, чтобы зарабатывать. Реальность:..."

#### 12. Сравнение
```
[Вариант A] vs [Вариант B]

[Плюсы и минусы каждого]

[Ваш вывод]

[Рекомендация]
```
**Пример:** "Продавать через контент vs через рекламу. Что выбрать?..."

---

### Категория: Question (Вовлечение)

#### 13. Вопрос-дилемма
```
[Ситуация, в которой нужно сделать выбор]

Вариант 1: [описание]
Вариант 2: [описание]

Что бы выбрали вы?
```
**Пример:** "У вас 100К на маркетинг. Куда вложить: в рекламу или в контент? Пишите в комментариях..."

#### 14. Опрос с историей
```
[Краткая история или контекст]

[Вопрос к аудитории]

Варианты:
А) [вариант]
Б) [вариант]

Пишите в комментариях!
```

#### 15. Вопрос-рефлексия
```
[Глубокий вопрос о работе/жизни]

[Почему это важно]

[Ваш ответ (опционально)]

А у вас как?
```
**Пример:** "Когда последний раз вы делали что-то, что вас пугало? Расскажите в комментариях..."

### Категория: Transformation (Трансформация)

#### 16. До/После с цифрами

```
- [метрика 1]: [значение]
- [метрика 2]: [значение]

ПОСЛЕ:
- [метрика 1]: [новое значение]
- [метрика 2]: [новое значение]

Что изменилось: [ключевое действие]
```

#### 17. Путь героя
```
Старт: [где был]
↓
Препятствие: [что мешало]
↓
Решение: [что сделал]
↓
Результат: [где сейчас]

Главный урок: [вывод]
```

## Прогревы

### 7-дневный прогрев (базовый)

| День | Цель | Тип поста | Смысл |
|------|------|-----------|-------|
| 1 | Боль | Провокационный вопрос | Показать проблему |
| 2 | Доверие | Личная история | Установить связь |
| 3 | Авторитет | Кейс клиента | Доказать экспертность |
| 4 | Возражения | FAQ или миф | Снять сомнения |
| 5 | Желание | Трансформация | Показать результат |
| 6 | FOMO | Ограничение | Создать срочность |
| 7 | Действие | Продающий | Дать CTA |

### 14-дневный прогрев (продвинутый)

| День | Тип | Фокус |
|------|-----|-------|
| 1-2 | Боль | Узнаваемая проблема |
| 3-4 | Контент | Бесплатная ценность |
| 5-6 | Авторитет | Кейсы, опыт |
| 7-8 | Engagement | Вопросы, опросы |
| 9-10 | Возражения | FAQ, мифы |
| 11-12 | Желание | До/После, трансформация |
| 13 | FOMO | Ограничение, дедлайн |
| 14 | Продажа | Оффер + CTA |

### 30-дневный прогрев (запуск)

**Неделя 1: Awareness**
- Дни 1-3: Боль (разные углы)
- Дни 4-5: Образовательный контент
- Дни 6-7: Engagement (вопросы)

**Неделя 2: Interest**
- Дни 8-10: Авторитет (кейсы, опыт)
- Дни 11-12: Ценность (инструменты)
- Дни 13-14: Истории успеха

**Неделя 3: Desire**
- Дни 15-17: Трансформация (до/после)
- Дни 18-19: Возражения (FAQ)
- Дни 20-21: Тизер продукта

**Неделя 4: Action**
- Дни 22-24: Анонс + детали
- Дни 25-27: Социальное доказательство
- Дни 28-29: FOMO + ограничения
- День 30: Финальный CTA

## Шаблоны по Тимочко

### Структура продающего вебинара

**5 задач вебинара (НЕ обучение!):**

| Блок | Время | Задача | Содержание |
|------|-------|--------|------------|
| **Intro** | 0-10 мин | Настроить на работу | Знакомство, программа, правила |
| **Вдохновение** | 10-60 мин | Вдохновить на тему | Актуальность, возможности, кейсы |
| **Доверие** | 60-90 мин | Создать доверие к эксперту | История, достижения, методология |
| **Путь** | 90-120 мин | Сформировать видение пути | 3-5 шагов, ошибки новичков |
| **Продукт** | 120-150 мин | Убедить в продукте | Программа, сравнения, УТП |
| **Продажа** | 150-180 мин | Продать продукт | Стек, цены, бонусы, ограничения |

### Структура Reels (40-60 сек)

```
СЕКУНДЫ 0-3: СЦЕПЛЕНИЕ
├── Заголовок-вопрос (читается на экране)
├── Цепляющая картинка/жест
└── Ни слова "привет, меня зовут"

СЕКУНДЫ 3-15: РАЗВИТИЕ  
├── Актуализация проблемы
├── "А знаешь, в чём дело?"
└── Лёгкая самопрезентация

СЕКУНДЫ 15-45: РЕШЕНИЕ
├── Конкретный ответ на вопрос
├── 1-3 шага/совета
└── Как это работает

СЕКУНДЫ 45-60: ФИНАЛ
├── Главный вывод одной фразой
├── CTA в середине (не в конце!)
└── Интрига для следующих роликов
```

### Топ-шаблоны Reels

#### 1. Вызов
```
Hook: "Спорим, ты не знал про эту фишку в Instagram?"
Развитие: "90% блогеров игнорируют это правило..."
Решение: "А секрет в том, что..."
Финал: "Попробуй и напиши в комментах результат"
```

#### 2. Инструкция
```
Hook: "Как найти 100 клиентов за неделю"
Развитие: "Все думают, что нужен большой бюджет..."
Решение: "На самом деле достаточно 3 шагов: [шаги]"
Финал: "Сохраняй, чтобы не потерять"
```

#### 3. Секрет
```
Hook: "Секрет, который скрывают топ-блогеры"
Развитие: "Почему у них миллионы просмотров, а у тебя сотни?"
Решение: "Всё дело в [конкретная техника]..."
Финал: "Кто попробует — отпишитесь в сторис"
```

#### 4. Страх потерь
```
Hook: "Ты всю жизнь продавал себя неправильно"
Развитие: "И поэтому теряешь 70% потенциальных клиентов..."
Решение: "Вместо [неправильно] делай [правильно]"
Финал: "Исправишь — доходы вырастут в разы"
```
