## Два типа SEO

### SEO для поиска
- Для "how-to" тем: "как постелить пол", "как настроить SEO"
- Видео находят через конкретные запросы
- 80-90% поисковых тегов

### SEO для алгоритмов
- Алгоритмы сами понимают, кому показывать
- Запуск в органику/рекомендации
- 60% тематических тегов, 40% поисковых

## Лимит тегов

**Всего:** 500 символов
**Цель:** Использовать 400-500 символов (80-100%)

❌ Использовать 180/500 = 36% → плохо
✅ Использовать 450/500 = 90% → хорошо

## Три типа тегов

### 1. Брендированные теги (2-4 шт)
```
Кто вы или что вы:
- Имя спикера
- Название студии/компании
- Название проекта

Дублировать в КАЖДОМ видео!
Цель: Связать все видео канала в единую экосистему
```

**Примеры:**
- Николай Вележанин
- Студия Вележанина
- Контент могущество

**На графике:** Низкий поиск + низкая конкуренция (левый нижний угол)

### 2. Тематические теги (6-8 шт)
```
Вектор движения видео:
- Большой объем поиска (40K-100K+)
- Высокая конкуренция — ОК
- Обычно 1-2 слова
- Примеры: "бизнес", "психология", "недвижимость"
```

**Примеры по нишам:**

| Ниша | Тематические теги |
|------|-------------------|
| Недвижимость | недвижка, квартиры, риэлтор, инвестиции |
| Психология | психология, счастье, саморазвитие, личностный рост |
| Бизнес | бизнес, мотивация, успех, предпринимательство |
| Строительство | строительство, стройка, дом, ремонт |

**На графике:** Высокий поиск + высокая конкуренция (правый верхний угол)

### 3. Поисковые теги (заполнить остаток до 500)
```
Как люди ищут:
- Начинаются с "как", "что такое", "зачем", "почему"
- Целевая зона: ВЫСОКИЙ поиск + НИЗКАЯ конкуренция
- Примеры:
  "как раскрутить канал на YouTube"
  "заработок в интернете без вложений"
  "как набрать миллион подписчиков"
```

**Отличие YouTube от Яндекс/Google:**
```
❌ Яндекс: "Купить квартиру Москва сейчас" (роботизировано)
✅ YouTube: "Где выгоднее купить квартиру?" (разговорно, как спросить друга)
```

**На графике:** Высокий поиск + НИЗКАЯ конкуренция (левый верхний угол) ⭐ ЗОЛОТАЯ ЗОНА

## Квадрантная модель тегов

```
         Высокая конкуренция
              ↑
    Q3 ❌     |     Q4 🟡
  (избегать)  |  (тематические)
              |
←─────────────┼─────────────→ Высокий поиск
              |
    Q1 🔵     |     Q2 ⭐
(брендированные)|  (ЗОЛОТО - поисковые)
              |
              ↓
         Низкая конкуренция
```

| Квадрант | Поиск | Конкуренция | Использование |
|----------|-------|-------------|---------------|
| Q1 | Низкий | Низкая | Брендированные, узкие ниши |
| Q2 ⭐ | ВЫСОКИЙ | НИЗКАЯ | Поисковые теги — ЦЕЛЬ |
| Q3 ❌ | Низкий | Высокая | Избегать |
| Q4 | Высокий | Высокая | Тематические теги |

## VidIQ — Основной инструмент

### Стоимость
- Платная: $8-12/месяц (рекомендуется)
- Бесплатная: ограничена (3-5 тегов за раз)

### Браузеры
- ✅ Google Chrome, Яндекс Браузер
- ❌ Safari, Edge, IE

### Метрики оценки тегов

| Метрика | Описание | Бенчмарки |
|---------|----------|-----------|
| **Поисковый объем** | Запросов/месяц | 100K+ отлично, 40K хорошо, <1K низко |
| **Конкуренция** | 0-100, как часто используют | 0-15 золото, 15-40 норма, 70+ избегать |
| **Общая оценка** | Объем + конкуренция | 65+ раньше хорошо, сейчас 73+ |
| **Относительная оценка** | Синергия с другими тегами | Чем выше — тем лучше работают вместе |

### Workflow поиска тегов
```
1. Вбить 3-5 seed-тегов (тема видео)
2. Просмотреть предложения VidIQ
3. Для каждого тега спросить:
   - Подходит ли теме видео?
   - Если зритель найдёт видео по этому тегу — кайфанёт?
   - В каком квадранте тег?
4. Выбрать теги из Q2 (высокий поиск + низкая конкуренция)
5. Скопировать в буфер обмена
6. Заполнить до 500 символов
```

### Итеративное расширение тегов
```
1. Начать с 3-5 seed-тегов
2. VidIQ предложит 10-20 связанных
3. Выбрать 5-8 лучших
4. Ввести их как новые seed
5. VidIQ предложит следующий слой
6. Повторять пока не заполнишь 500 символов
```

### Анализ конкурентов
- Зелёная иконка = видео использует этот тег
- Клик → смотреть полную стратегию конкурента
- Вопрос: "Почему это видео успешно с этим тегом?"

## Таймкоды

### Обязательные правила
```
1. НАЧАЛО С 0:00 — иначе не работают!
2. Разрыв между таймкодами: минимум 10 секунд
3. Можно в описании ИЛИ в комментариях
```

### Создание через нейросеть

**Шаги:**
1. Загрузить видео на YouTube (доступ по ссылке)
2. Транскрибировать через TurboScribe
3. Скачать с временными метками
4. Загрузить в ChatGPT/Claude
5. Дать промпт

**Промпт:**
```
Привет! Напиши таймкоды, чтобы людям было удобно
смотреть видео.
Пример таймкодов: [скриншот]
Это очень важная задача.
```

**⚠️ ВАЖНО:** Всегда проверять — нейросети врут!

## Структура описания

### Принцип
**"Размазываем теги как масло по хлебу"** — каждый тег должен появиться в описании естественно

### Шаблон описания
```
Всем привет, с вами [БРЕНД 1], [БРЕНД 2].

Вы на канале про [БРЕНД 3 / НАЗВАНИЕ КАНАЛА].

Сегодня мы затронем важную тему [ТЕМАТИЧЕСКИЙ 1] и [ТЕМАТИЧЕСКИЙ 2].

Поговорим о том, как добавить [ТЕМАТИЧЕСКИЙ 3] и [ТЕМАТИЧЕСКИЙ 4] в свою жизнь.

А самое главное, выясним, [ПОИСКОВЫЙ 1].

Более того, расскажем, [ПОИСКОВЫЙ 2] и [ПОИСКОВЫЙ 3].

А самое главное — [ПОИСКОВЫЙ 4].

[CTA на лайк/подписку/комментарий]
```

### Что НЕ делать
```
❌ Спам тегов: "Счастье. Счастье как обрести, обрести счастье как..."
   → YouTube пенализирует за keyword stuffing

✅ Теги разделены несколькими словами естественного текста
```

### Ссылки
- На новых каналах ссылки в описании неактивны
- Ссылки в закрепленном комментарии РАБОТАЮТ всегда
- Дублировать важные ссылки в комментарий
- Максимум 1-2 ссылки вверху описания

## ChatGPT + TurboScribe Workflow

### Полный процесс написания SEO

**Шаг 1: Транскрибация**
```
1. Загрузить видео на YouTube (доступ по ссылке)
2. Скопировать URL видео
3. TurboScribe → вставить URL
4. Настройки: режим Kit, русский язык
5. ✅ Включить "Временные метки"
6. Расширенный экспорт → .docx с таймкодами
```

**Шаг 2: Настройка ChatGPT**
```
1. Создать НОВЫЙ проект в ChatGPT
2. Назвать (например "YouTube SEO")
3. Загрузить промпт с инструкциями
4. Сохранить для повторного использования
```

**Шаг 3: Генерация**
```
1. Открыть сохранённый проект
2. Загрузить .docx транскрибацию
3. Отправить
4. Ждать 30-60 сек
```

**Шаг 4: Что генерирует ChatGPT**
| Элемент | Качество | Действие |
|---------|----------|----------|
| Названия | ❌ Плохо | Переписать вручную по ВИСП |
| Описание | 🟡 50/50 | Проверить, отредактировать |
| Таймкоды | ✅ Хорошо | Проверить временные метки |
| Теги | 🟡 50/50 | Проверить ВСЕ через VidIQ |

**Шаг 5: Верификация тегов**
```
1. Скопировать теги из ChatGPT
2. Вставить в VidIQ
3. Отбросить теги из Q3 (низкий поиск + высокая конкуренция)
4. Оставить теги из Q2 (высокий поиск + низкая конкуренция)
5. Вернуть проверенные теги в ChatGPT
6. Попросить переписать описание с этими тегами
```

**Шаг 6: Доработка таймкодов**
```
Промпт: "Переделай таймкоды, чтобы хотелось сразу кликнуть"

До: "06:00 Представление Кати"
После: "06:00 Как Катя заработала на Paywall"
```

### Важно
```
⚠️ ChatGPT — помощник, не решение
⚠️ 50% тегов ChatGPT = мусор
⚠️ 100% нужно проверять через VidIQ
⚠️ Таймкоды могут сдвигаться на 10-15 сек
```

### Сравнение методов

| Метод | Время | Качество |
|-------|-------|----------|
| Ручной (2016-2018) | 2-6 часов | Высокое |
| Голосовые сообщения | 5 минут | 50% |
| ChatGPT + VidIQ | 15-20 минут | Высокое |

## Anti-patterns

### Критические ошибки

| Ошибка | Проблема | Решение |
|--------|----------|---------|
| Таймкоды не с 0:00 | Не работают вообще | Первый таймкод = 00:00 |
| Разрыв < 10 сек | Глава не отобразится | Минимум 10 секунд между |
| Нет тегов канала | Видео не связываются | Добавить 2-4 брендированных |
| 180/500 символов | 36% эффективности | Заполнить до 450-500 |
| Keyword stuffing | Пенализация YouTube | Теги через естественный текст |
| Q3 теги | Много конкурентов, мало поиска | Целиться в Q2 |
| SEO-название | Убивает CTR | Сначала ВИСП, потом SEO |
| Копировать ChatGPT | 50% мусора | Проверять ВСЁ через VidIQ |
| Не проверять таймкоды | Сдвиг на 10-15 сек | Сверять с видео вручную |
| Теги без описания | Потеря SEO | Каждый тег — в описание |

### Кейс: Сила тегов канала
```
Канал: 10K подписчиков за 6 месяцев (слабо)
Проблема: НОЛЬ тегов канала
Действие: Добавили теги канала
Результат: Первое видео после → 300K просмотров
Причина: Видео зацепилось за похожие через общий тег
```

## Чек-лист SEO

### Теги
- [ ] Брендированные: 2-4 шт
- [ ] Тематические: 6-8 шт
- [ ] Поисковые: заполнить до 500 символов
- [ ] Использовано 400-500 символов (80-100%)
- [ ] Все теги проверены через VidIQ
- [ ] Поисковые из Q2 (высокий поиск + низкая конкуренция)

### Описание
- [ ] Каждый тег появляется в тексте
- [ ] Теги разделены естественным текстом
- [ ] Нет keyword stuffing
- [ ] CTA включён
- [ ] Максимум 1-2 ссылки вверху

### Таймкоды
- [ ] Первый таймкод = 00:00
- [ ] Разрыв между таймкодами 10+ сек
- [ ] Названия вызывают желание кликнуть
- [ ] Временные метки сверены с видео

### Название
- [ ] Сначала ВИСП, потом SEO
- [ ] 1-2 тематических тега вплетены естественно
- [ ] Не роботизированное

### Общее
- [ ] Теги канала настроены (2-4 брендированных)
- [ ] Ссылки продублированы в закреплённый комментарий
