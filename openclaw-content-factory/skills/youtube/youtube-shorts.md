---

## Специфика Shorts

| Параметр | Значение |
|----------|----------|
| Длина | 15-60 секунд |
| Критические секунды | Первые 3 |
| Формат | Вертикальный 9:16 |
| Цель | Hook → Payoff → (опционально) CTA |

---

## Структура Short

```
HOOK (3 секунды)
↓
РАЗВИТИЕ (основной контент)
↓
PAYOFF / ОТВЕТ (в конце)
```

**Правило:** Если есть ответ на вопрос — держи интригу до конца.

---

## ВИСП для Shorts

Первые 3 секунды должны содержать минимум 2 компонента:

| Приоритет | Компоненты |
|-----------|------------|
| 1 | Причастность + Интрига |
| 2 | Выгода + Срочность |

**Пример:**
```
"Если ты давишь комедоны каждый вечер — вот секрет..."
- Причастность: "если ты давишь комедоны"
- Интрига: "вот секрет"
```

## Нарезка из длинных видео

### Что хорошо режется:
- Отдельные payoffs с сильным началом
- Вопросы из-за кадра + ответы
- Истории с punch line
- Списки (1 пункт = 1 short)

### Что плохо режется:
- Контекстно-зависимые моменты
- Длинные объяснения
- Секции без clear payoff

---

## Формула вирусного Short

```
1. HOOK: Неожиданное утверждение / вопрос (3 сек)
2. TENSION: Развитие, примеры (20-40 сек)
3. PAYOFF: Ответ / punch line (последние 5 сек)
```

---

## Retention в Shorts

**Проблема:** Зритель листает за 1-2 секунды

**Решение:**
- Первый кадр = самый интересный
- Текст на экране с первой секунды
- Голос начинается сразу (без пауз)
- Динамика с первого момента

## 10 жирных техник шортсов

### 1. Топ-список
> "3 место, 2 место, 1 место" — от менее шокирующего к самому жирному

**Правила:**
- Никогда не говорить слово "топ" — это привет из 2000-х
- Порядок: 3 → 2 → 1 (не наоборот!)
- Третье место тоже должно быть жирным
- Тема должна попадать в 200 миллионов (широкий охват)

```
❌ "Топ-3 мебели в офис" — узко, нахуй никому не надо
✅ "1 января тебе будет плохо от этих напитков" — попадает в каждого
```

**Ключ:** Самое сильное слово — в начало предложения!
```
❌ "Топ-3 алкоголя, от которых тебе будет плохо 1 января"
✅ "1 января тебе будет максимально плохо от этих напитков"
```

---

### 2. Никогда нельзя
> Экспрессивный категоричный запрет

**Требования:**
- Спикер ДОЛЖЕН говорить экспрессивно (монотонно = провал)
- Нужна харизма и эмоция
- Часто совмещается с топ-списком

```
"Никогда не говори по телефону эти три вещи!"
"Никогда не покупай эти акции!"
"Никогда не иди с ним на свидание, если он написал тебе это."
```

---

### 3. Представьте
> Погружение в образ, который есть в голове у каждого

**Правило:** Образ должен быть таким, в котором россиянин 100% БЫЛ, БУДЕТ или БОИТСЯ оказаться.

```
"Представьте, вы просыпаетесь от громкого стука в дверь.
Подходите — оттуда голос: 'Откройте, полиция!' Что делать?"

"Представьте, вам идти на свидание, а у вас крутило живот."

"Представьте, вы надеваете свои любимые джинсы-скинни —
а они вам маленькие."
```

---

### 4. Образ знакомый каждому (Жиза)
> То, что было у КАЖДОГО — ностальгия, общий опыт

```
"А вас в детстве тоже родители мучили ингаляциями над картошкой?"
"А вы тоже дышали над картошкой в детстве?"
"А вы тоже в детский сад ходили с жёлтой коробочкой от киндер-сюрприза с чесноком?"
"А вы помните, как пахнет лес после дождя?"
```

**Тест:** Если >80% людей поднимут руку "да" — это жиза.

**Важно:** Попадать в УНИВЕРСАЛЬНЫЙ опыт, не в узкую нишу.
```
❌ "Вас тоже заебало, что в приложении ВТБ..." — узко (не у всех ВТБ)
✅ "Бывший написал 'Привет, я соскучился' — что делать?" — универсально
```

---

### 5. Присоска
> Привязка к бренду/приложению, которое есть У КАЖДОГО

**Бренды для присоски:**
- Сбербанк, Тинькофф, ВТБ (банки)
- Яндекс, Госуслуги (сервисы)
- Пятёрочка, Ашан, Магнит (магазины)
- Авито, Wildberries, Ozon (маркетплейсы)
- Nike, Adidas (бренды одежды)
- Apple, iPhone (техника)

```
"У вас на телефоне есть приложение Сбербанк?
Вы знали, что там можно настроить автокопилку?"

"В Пятёрочке есть потрясающая закуска, которая безвредна для организма..."

"Знаете, на чём зарабатывает Лукойл?"

"Apple сейчас разрабатывают свои автомобили — знали?"
```

---

### 6. Загадка
> Вопрос → несколько смешных неправильных ответов → правильный ответ

**Формат:**
1. Один человек задаёт вопрос
2. Второй (или сам спикер) даёт "тупые" ответы
3. Правильный ответ в конце

```
— От чего зависит женский оргазм?
— От белья?
— Нет.
— От запаха мужчины?
— Нет.
— От выключенного света?
— Нет! От того, насколько она чувствует себя желанной.
```

**Лайфхак:** Играть роль "конченого долбоёба" с нелепыми ответами — удерживает внимание.

---

### 7. Спорное мнение / Провокация
> Противоречие общепринятому мнению

```
"Насрать на визуальные хуки."
"Тинькофф круче, чем Сбербанк."
"Никогда не надо постить шортсы на один канал."
```

**Эффект:** Комментарии, споры, вовлечение.

---

### 8. Волшебная таблетка
> Одно действие → много результатов

```
"Одна капля — и все женщины твои."
"Одно упражнение от ожирения, запоров и головной боли."
"Одна фраза, которая изменит ваши отношения."
```

---

### 9. Самый худший
> Негатив цепляет лучше позитива

```
"Самая худшая водка среди дорогих."
"3 района Москвы, где жить не советую."
"Это продукт, от которых ты точно умрёшь 1 января."
```

---

### 10. История
> Мини-сторителлинг с punch line

```
"Короче, была у меня подруга. 7 лет встречалась с парнем.
И вот они женятся. Через две недели звонит мне:
'Мы разведёмся.' Знаете почему?..."
```

**Структура:** Setup → Конфликт → Неожиданный финал

## Критические правила шортсов

### Тест на 200 миллионов
> Шорт ВСЕГДА должен попадать в максимально широкую аудиторию

```
❌ "Почему не покупают на Авито" — только для продавцов на Авито
❌ "Стоимость этих тачек вырастет после Нового года" — только для тех, кто хочет купить тачку
✅ "Три способа сэкономить на покупке авто" — для всех, кто когда-либо будет покупать
✅ "Как заработать на ненужных вещах дома" — у каждого есть хлам
```

### Правило первой секунды
> Самое сильное слово — в самое начало

```
❌ "Топ-3 алкоголя, от которых тебе будет плохо 1 января"
✅ "1 января тебе будет максимально плохо от этих напитков"
```

### Правило третьего места
> В топ-списке третье место тоже должно быть жирным

Если третье место слабое — зритель отвалится, не дождавшись первого.

---

## Типы успешных Shorts (базовые)

1. **Ответ на вопрос** — интрига до конца
2. **Quick tip** — одна полезная идея
3. **Мини-история** — setup → conflict → resolution
4. **Разрушение мифа** — "Все думают X, но на самом деле Y"
5. **Before/After** — трансформация

---

## Anti-patterns

- Длинное intro (даже 2-3 секунды = много)
- "Всем привет" — сразу свайп
- Ответ в начале (нет причины смотреть)
- Монотонный темп
- Нет визуальной динамики

---

## Чек-лист Short

- [ ] Первые 3 секунды цепляют
- [ ] Нет приветствия/прощания
- [ ] Причастность + Интрига с первого кадра
- [ ] Payoff/ответ в конце (не в начале)
- [ ] Текст на экране
- [ ] Динамичный темп

## Монетизация Shorts — 11 способов

### Типология шортсов

| Тип | Описание | Что делать |
|-----|----------|------------|
| **В кассу** | Есть длинное видео по той же теме | Закрывать на видео |
| **Мимо кассы** | Нет подходящего видео | Закрывать на лид-магнит |

**Пример "в кассу":**
- Шортс: "5 фраз, чтобы он тебя боготворил"
- Длинное видео: "5 фраз для отношений" → ОК

**Пример "мимо кассы":**
- Шортс: "Правда про зубы Моргенштерна"
- Длинное видео: "Как я сделал виниры" → НЕ ОК (убьёт удержание)

---

### 11 способов закрытия на целевое действие

#### 1. Заглушка / Пэкшот
```
Внутрь шортса вставляем превью длинного видео
+ текст "Смотри полное видео в описании"
+ ссылка в описании
```
**Когда:** Шортс "в кассу"
**Важно:** Под заглушкой должна идти аудиодорожка с триггерными словами

#### 2. Подарок в описании
```
Закреплённый комментарий: "Забери подарок тут 🎁"
→ Ведёт на видео с лид-магнитом
→ В видео: "Спасибо что перешёл, по ссылке в описании забирай подарок"
```
**Когда:** Шортс "мимо кассы"

#### 3. QR-код / Ссылка в комментариях
```
⚠️ НЕ РЕКОМЕНДУЕТСЯ — работает плохо
- QR-код: надо заскринить → перейти в ленту → сканировать
- Ссылка в комментах: не кликабельная
```
**Если используешь:** "Скопируй ссылку и перейди по ней"

#### 4. Техника "1-2-3-4, а 5 в полном видео"
```
"3 упражнения для спины:
1. [жирный совет]
2. [жирный совет]
А 3-е и 4-е упражнения — в полном видео по ссылке"
```
**Риск:** Хейт в комментах ("впадлу записать?")
**Правило:** Первые пункты ДОЛЖНЫ быть жирными, с реальной пользой

#### 5. Прямой призыв (Call to Action)
```
"Забери подарок по ссылке в описании"
"Переходи по ссылке в шапке профиля"
```

#### 6. Подписка на канал

**Где размещать:**

| Место | Как делать |
|-------|------------|
| Начало | После ВИСП + актуализации проблемы (5-10 сек) |
| Середина | После первого жирного контента |
| Конец | С ВИСПОМ! ("Здесь про самые дешёвые способы сохранить зубы") |

**Почему после актуализации:**
```
❌ "Как вылечить кариес дома — подписывайся" → прослушают мимо
✅ "Как вылечить кариес дома... [5 сек актуализации]...
    Мой канал про здоровье зубов без больших трат — подписывайся"
```

#### 7. Графическая плашка подписки
```
Плашка + стрелка на кнопку "Подписаться"
+ ТЕКСТ с ответом "ЗАЧЕМ подписываться?"

Примеры текста:
- "Подпишись, чтобы собирать урожай вёдрами"
- "Подпишись, чтобы не потерять свой бизнес"
- "Идеальная натуральная аптечка — здесь"
```
**Усиление:** Звук, дым-эффект, анимация

#### 8. ТГ-канал на экране
```
Собачка + название канала (простое, без @)
Или: "ТГК [название]"
Или: Скриншот телеграм-канала
```
**Размещение:** Уголок экрана, не на весь экран (отвал)

#### 9. Ватермарка / Логотип
```
Логотип компании на всех шортсах
→ Узнаваемость бренда
→ Косвенное влияние на продажи
```
**Must-have для:** Франшизы, оффлайн-бизнесы с длинным циклом сделки

#### 10. Номер телефона
```
Номер на экране
→ Больше доверия ("сам позвоню, не накуканят")
```
**Для:** Услуги, консультации, продажа авто

#### 11. Промокод
```
ОНЛАЙН:
"По промокоду [X] получи книгу бесплатно — ссылка в шапке профиля"

ОФФЛАЙН:
"Сделай скриншот этого промокода — покажи в клинике, получи 20% скидку"
```

---

### Лестница Ханта для шортсов

```
Холодный трафик (широкий охват):
→ Ватермарка, подписка на канал

Тёплый трафик (интерес к теме):
→ ТГ-канал, лид-магнит

Горячий трафик (готов к действию):
→ Номер телефона, промокод, прямая ссылка
```

**Правило:** Нельзя перепрыгивать ступени. Шортс на миллион просмотров = холодная аудитория.

---

### Перезалив шортсов с воронкой

**Методика:**
1. Взять самый популярный шортс канала
2. Добавить элемент воронки (плашка, пэкшот, CTA)
3. Перезалить
4. Замерить конверсию до/после

**Что добавлять:**
- Графическая плашка с триггерным текстом
- Пэкшот на длинное видео (если "в кассу")
- CTA на подписку с ВИСПом

---

### Чек-лист монетизации шортса

- [ ] Определён тип: "в кассу" или "мимо кассы"
- [ ] Выбран способ закрытия по лестнице Ханта
- [ ] Текст CTA отвечает на "Зачем?"
- [ ] CTA по ВИСПу (даже в конце)
- [ ] Элемент воронки не убивает удержание
