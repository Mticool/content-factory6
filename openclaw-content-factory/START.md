# 🚀 START — Фабрика Контента

> Ты — AI-агент (Claude Code / OpenClaw). Человек положил этот файл в твою рабочую папку.
> Прочитай его и выполни инструкции. После этого ты будешь уметь создавать контент.

---

## Что это

Фабрика Контента — система из 30 скиллов для создания контента. Посты, видео, карусели, рилсы — под голос и стиль твоего эксперта.

**Твоя задача:** развернуть систему и провести интервью с экспертом.

---

## Шаг 1: Проверь структуру

Рядом с этим файлом должны быть папки:
```
skills/          ← 30 скиллов (методики контента)
projects/        ← шаблон проекта + пример
agents/          ← описания агентов
docs/            ← документация
```

Если папки на месте — переходи к Шагу 2.
Если нет — скажи человеку: "Нужна полная папка конструктора, не только START.md"

---

## Шаг 2: Создай проект

```bash
cp -r projects/_template projects/{имя}
```

Спроси человека: **"Как назвать проект? (одно слово, латиницей — например: daria, coach, doctor)"**

---

## Шаг 3: Проведи интервью

Прочитай файл `projects/{имя}/onboarding/interview.md` — там скрипт интервью из 7 блоков.

**Правила:**
- Один вопрос за раз
- Уточняй если ответ поверхностный
- Записывай ответы сразу в файлы brand/

**Результат интервью → файлы:**
- `brand/profile.md` — кто эксперт
- `brand/audience.md` — для кого
- `brand/voice-style.md` — как писать

---

## Шаг 4 (опционально): Распаковка личности

После интервью предложи: **"Хочешь глубже распаковать себя? Это 40-60 минут, и контент будет звучать как ТЫ."**

Если да — прочитай `skills/personal-unpacking/SKILL.md` и проведи 8 блоков.

**Результат → файл:**
- `brand/personal-dna.md` — суперсилы, ценности, истории, архетип

---

## Шаг 5: Первый контент

После интервью сгенерируй 3 тестовых поста:

1. Прочитай `skills/threads/SKILL.md` (или другой формат, который нужен человеку)
2. Прочитай всю папку `brand/` — это голос эксперта
3. Сгенерируй 3 поста
4. Покажи человеку, спроси: "Похоже на тебя? Что поправить?"

---

## Как работать дальше

### Генерация контента
Когда человек просит контент:
1. Определи формат по запросу (см. таблицу ниже)
2. Прочитай скилл из `skills/{формат}/SKILL.md`
3. Прочитай `brand/` — голос, аудитория, ДНК
4. Прочитай `learning/` — что работало раньше
5. Генерируй

### Таблица форматов

| Запрос | Скилл |
|--------|-------|
| пост, threads, тредс | `skills/threads/` |
| youtube, ютуб, скрипт | `skills/youtube/` |
| reels, рилс, shorts | `skills/reels/` |
| карусель, carousel | `skills/carousel/` |
| telegram, тг пост | `skills/telegram/` |
| распаковка личности | `skills/personal-unpacking/` |
| распаковка смыслов, 8 столбиков | `skills/customer-research/` |
| оффер, offer | `skills/offer/` |
| запуск, launch, прогрев | `skills/launch/` |
| картинка, обложка | `skills/nano-banana/` |
| история, кейс | `skills/storytelling/` |
| продающий, CTA | `skills/selling-meanings/` |
| заголовок, хук | `skills/headlines/` |
| swipefile, разбор видео | `skills/swipefile/` |

### Обучение
Когда человек говорит "это сработало" → запиши в `learning/approved-patterns.md`
Когда "это не сработало" → запиши в `learning/anti-patterns.md`
