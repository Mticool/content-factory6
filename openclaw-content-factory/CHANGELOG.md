## v1.21 (2026-03-06)

### 🏗️ Реструктуризация скиллов: 32 → 10

Полная переработка архитектуры скиллов. Вместо 32 отдельных скиллов — 10 мега-скиллов с умным роутингом.

| # | Скилл | Процесс |
|---|-------|---------|
| 1 | ✍️ `writing` | Тексты для соцсетей (threads, telegram, копирайтинг, редактура, заголовки, смыслы) |
| 2 | 🎬 `video-scripts` | Сценарии видео (YouTube, Reels/Shorts, сторителлинг) |
| 3 | 🎠 `carousel` | Instagram карусели |
| 4 | 🖼️ `image-gen` | Генерация картинок (промпты, identity-kit, обложки) |
| 5 | 🎥 `video-gen` | Генерация видео (Veo, HeyGen, таймлапсы, монтаж) |
| 6 | 🧭 `strategy` | Распаковка, CustDev, оффер, запуски |
| 7 | 🔍 `research` | SwipeFile, обработка обучающих материалов |
| 8 | 🔎 `seo` | Статьи для блога, YouTube-упаковка |
| 9 | 🤖 `agent-builder` | Создание агентов и скиллов |
| 10 | ⚙️ `system` | Память, настройки, API |

### Что изменилось
- **File Router** — каждый SKILL.md содержит таблицу маршрутизации: агент читает только нужную подпапку, не всё
- **Context Loading** — стандартный блок: каждый скилл автоматически подтягивает brand/voice-style.md, brand/audience.md, learning/approved-patterns.md
- **Меньше токенов** — вместо 32 описаний в промпте теперь 10
- **Удалены дубли** — offer-core (= offer), creator (= agent-architect)
- **Убран хардкод** — никаких абсолютных путей, всё через {baseDir}

### Обновление
- `update.sh` теперь чисто удаляет старые скиллы перед установкой новых (бэкап создаётся автоматически)
