## 🎬 AI Video Generation (Veo 3 + конкуренты)

### Ключевые тренды
- **Production-ready качество**: Veo 3.1, Sora 2, Kling 3.0, Seedance достигли уровня профессионального видео
- **Вертикальный формат доминирует**: 9:16 уже 43.7% и обгоняет 16:9 в 2026
- **Скорость генерации**: с 1-3 минут до 10-30 секунд к концу года
- **Long-Form Mode**: Veo готовит 10-минутные нарративы с консистентностью
- **Image-to-video**: новички начинают с текста, опытные переходят на image-guided для контроля
- **Native audio**: Veo 3 генерирует видео со звуком из коробки

### Применимо для OzonCheck (селлеры маркетплейсов)

✅ **Карточки товара с видео**
- Генерация product demo за 30 секунд вместо съёмки
- Промпт: "Вертикальное видео 9:16, продукт [название] на столе, камера медленно приближается, студийный свет, 8 секунд"
- Применение: заменить статичные фото на Ozon/WB на AI-видео

✅ **Кейсы "до/после"**
- Image-to-video: взять фото продукта → добавить движение
- Промпт: "Таймлапс распаковки товара, вертикальное видео, 10 секунд"
- Применение: Reels/Shorts с распаковкой без съёмочной группы

✅ **Обучалки для продавцов**
- Long-Form Mode для 10-минутных инструкций "Как настроить рекламу на Ozon"
- Veo 3 с native audio = озвучка без TTS отдельно
- Применение: YouTube контент для канала OzonCheck

✅ **A/B тесты креативов**
- Генерация 5 вариантов видео за 5 минут
- Тест в рекламе ВК/Телеграм, выбрать лучший
- Применение: снизить стоимость креативов в 10 раз

### Новые форматы видео

🔥 **AI Product Reveal**
- Генерация "кинематографичной" презентации товара без студии
- Пример: Veo 3 создаёт видео "профессионального качества, неотличимое от снятого"

🔥 **Vertical-First Content**
- Вертикальное видео теперь не "адаптация", а основной формат
- 9:16 обгоняет 16:9 → все креативы сразу вертикальные

🔥 **Real-Time Video Ads**
- 10-30 секунд генерации = можно создавать видео в режиме реального времени под акции
- Пример: "Скидка началась → промпт → видео через 20 сек → запуск рекламы"

## 📱 YouTube Shorts / Reels / TikTok Strategy

### Ключевые тренды
- **YouTube Shorts = 2B пользователей** (обогнал TikTok 1.59B)
- **Вирусность не ограничена 48 часами**: алгоритм дал больше времени на раскрутку
- **AI-generated creator likenesses**: YouTube планирует AI-аватары для Shorts
- **Repurpose-стратегия**: не нужны новые идеи каждый раз — переупаковка работает
- **Shareability = новый ключевой фактор**: Instagram ценит "sends per reach" (пересылки в DM)
- **Retention важнее просмотров**: досмотры = алгоритм продвигает

### Применимо для OzonCheck

✅ **Repurpose контента**
- Одна идея → 3 формата (YouTube Shorts, Reels, TikTok)
- Пример: "Топ-3 ошибки селлеров на Ozon" → снять 1 раз, порезать на 3 платформы
- Применение: экономия времени, максимум охвата

✅ **Фокус на удержание**
- Структура: хук (0-3 сек) → проблема → решение → CTA
- Пример: "Почему твои товары не продаются? [пауза] Проверь эти 3 параметра в OzonCheck"
- Применение: увеличить досмотры = больше охвата

✅ **"Sends per Reach" для Reels**
- Создавать контент, которым хочется поделиться с другими селлерами
- Пример: "Калькулятор рентабельности за 30 секунд" → делятся в чатах продавцов
- Применение: виральность через пересылки

✅ **YouTube Shorts как воронка**
- 2B пользователей = огромная аудитория для холодного трафика
- Shorts → профиль → полное видео → сайт OzonCheck
- Применение: бесплатный трафик в сервис

### Новые форматы видео

🔥 **AI-generated Shorts**
- YouTube готовит AI-аватары для автоматизации Shorts
- Пример: скрипт → AI-аватар → готовое видео без съёмки
- **Применимо сейчас**: HeyGen/Synthesia для экспертного контента OzonCheck

🔥 **Extended Virality Window**
- Shorts могут выстрелить через неделю/месяц, не только в первые 48 часов
- Применение: не удалять "непопулярные" видео — дать время

🔥 **Vertical Native Storytelling**
- Не "горизонтальное видео повернули", а вертикальный сторителлинг с нуля
- Пример: не показывать экран компьютера, а использовать полный вертикальный кадр

## 🤖 AI Avatar Video Marketing

### Ключевые тренды
- **Реализм вырос 2023-2026**: топ-платформы создают "highly realistic" аватары
- **Два направления**: UGC-focused (соцсети, бренды) vs корпоративные (обучение)
- **Персонализация кампаний**: масштабирование видео без потери качества
- **Лидеры рынка**: Synthesia (#1 enterprise), HeyGen, Creatify Aurora, Arcads, Superscale

### Применимо для OzonCheck

✅ **Экспертный контент без камеры**
- AI-аватар эксперта OzonCheck объясняет аналитику
- Пример: "Меня зовут [Имя], я эксперт OzonCheck. Сегодня расскажу про юнит-экономику"
- Применение: видео-курсы, FAQ, обучалки без съёмок

✅ **Персонализация для B2B**
- Создать аватар основателя OzonCheck → персонализированные приветствия для новых клиентов
- Пример: "Привет, [Имя]! Я Марат, основатель OzonCheck. Вижу, ты только зарегистрировался..."
- Применение: повысить engagement в email/Telegram кампаниях

✅ **Масштабирование FAQ**
- Один аватар → 50 FAQ-видео за день
- Пример: скрипты частых вопросов → AI-аватар озвучивает → база знаний OzonCheck
- Применение: снизить нагрузку на поддержку

✅ **UGC-стиль для соцсетей**
- UGC-focused аватары выглядят как реальные селлеры
- Пример: "Я селлер на Ozon, использую OzonCheck 6 месяцев, вот мой результат..."
- Применение: социальное доказательство без найма актёров

### Новые форматы видео

🔥 **Custom Brand Avatars**
- Создать собственного аватара бренда OzonCheck (не стоковый)
- Пример: Avatar IV от HeyGen — максимальная реалистичность с мимикой/жестами
- **Реально сейчас**: Synthesia Enterprise, HeyGen custom avatar

🔥 **Photo-to-Avatar**
- Одно фото → аватар (HeyGen, Creatify)
- Пример: фото эксперта OzonCheck → аватар для видео
- Применение: быстрый запуск без кастомной съёмки

🔥 **AI-аватары для продаж**
- Sales-focused аватары (Relevance AI — бесплатный)
- Пример: персонализированные видео-предложения для крупных клиентов
- Применение: outbound-кампании для B2B (агентства, консультанты)

## 🛒 Short-Form Video для E-commerce / Маркетплейсов

### Ключевые тренды
- **Shoppable videos**: покупка прямо из видео (встроенные кнопки, ссылки)
- **Вертикальное видео выходит на лендинги**: не только соцсети, но и сайты товаров
- **Интерактивность**: polls, chats, Q&A прямо в видео
- **Мобильная оптимизация критична**: большинство покупок с телефона
- **Bite-sized вертикальные видео**: формат TikTok/Reels для карточек товаров

### Применимо для OzonCheck (селлеры маркетплейсов)

✅ **Shoppable Shorts для товаров**
- Видео товара + кнопка "Купить на Ozon/WB"
- Пример: "Видео распаковки → клик → переход в карточку на маркетплейсе"
- Применение: встроить видео на лендинги селлеров, которые используют OzonCheck

✅ **Вертикальное видео в карточках товара**
- Загрузить вертикальное видео в карточку на Ozon/WB (не горизонтальное)
- Пример: 9:16 видео "как использовать продукт" = выше конверсия
- Применение: рекомендация для клиентов OzonCheck

✅ **Видео-отзывы покупателей**
- Короткие вертикальные отзывы (30 сек) вместо текста
- Пример: "Заказал через OzonCheck, вот результат за месяц [показ экрана]"
- Применение: UGC-контент для социального доказательства

✅ **Интерактивные видео на сайте**
- Встроить видео с polls/quiz на лендинг OzonCheck
- Пример: "Какой тариф вам подходит? [опрос в видео]"
- Применение: увеличить engagement на сайте

✅ **Вертикальные видео на лендингах**
- Не только соцсети — встроить вертикальное видео прямо на главную OzonCheck
- Пример: вместо баннера — авто-play вертикальное видео "Что такое OzonCheck за 10 секунд"
- Применение: повысить конверсию лендинга

### Новые форматы видео

🔥 **Shoppable Product Demos**
- Видео товара с кликабельными зонами (клик на продукт → переход на покупку)
- Пример: видео витрины товаров → клик → карточка маркетплейса
- **Платформы**: Firework, Tolstoy, Taggbox

🔥 **Vertical Landing Page Videos**
- Вертикальное видео как центральный элемент лендинга
- Пример: главная OzonCheck = вертикальное видео на весь экран + CTA
- Применение: захватить мобильный трафик (большинство пользователей)

🔥 **Social-to-Site Video Sync**
- Автоматически импортировать Instagram/TikTok видео на сайт
- Пример: селлер постит Reels → автоматически появляется на лендинге его бренда
- **Платформы**: Firework (интеграция с Adobe Commerce, WooCommerce)

## 🎯 Конкретные идеи для видео OzonCheck

### Форматы для немедленного запуска

1. **"OzonCheck за 10 секунд" (AI-генерация)**
   - Veo 3: вертикальное видео, экран с дашбордом, камера зумит на ключевые метрики
   - Формат: YouTube Shorts, Reels, TikTok
   - Цель: холодный трафик

2. **"Топ-3 ошибки селлеров" (AI-аватар)**
   - HeyGen/Synthesia: аватар эксперта перечисляет ошибки + решение через OzonCheck
   - Формат: 30 сек вертикальное
   - Цель: образовательный контент + лид-магнит

3. **"До/После использования OzonCheck" (Image-to-video)**
   - Скриншот показателей "до" → AI-анимация → скриншот "после"
   - Формат: 15 сек, вертикальное
   - Цель: социальное доказательство

4. **"FAQ: как работает юнит-экономика?" (AI-аватар + Shoppable)**
   - Серия из 10 FAQ-видео с аватаром эксперта
   - Встроить на сайт OzonCheck с кнопкой "Попробовать бесплатно"
   - Формат: 20-30 сек, вертикальное
   - Цель: база знаний + конверсия

5. **"Калькулятор рентабельности за 30 секунд" (Screen recording + AI-voice)**
   - Запись экрана OzonCheck + AI-озвучка
   - Формат: вертикальное, 30 сек
   - Цель: виральность через "sends per reach" (делятся в чатах селлеров)

6. **"Кейс селлера: +300% прибыли" (Repurpose-стратегия)**
   - Одно интервью селлера → порезать на 5 коротких Shorts
   - Каждый Shorts = отдельный инсайт
   - Формат: вертикальное, 30-60 сек
   - Цель: социальное доказательство + органический охват

7. **"Вертикальный лендинг для мобильных" (Native vertical)**
   - Главная страница OzonCheck для мобильных = вертикальное видео на весь экран
   - Авто-play: "Привет! Я OzonCheck, твой финансовый аналитик для маркетплейсов..."
   - Формат: 10 сек, зацикленное
   - Цель: повысить конверсию мобильного трафика
