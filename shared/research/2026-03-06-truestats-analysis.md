## 1. ОБЗОР КОНКУРЕНТА: TrueStats

### 1.1 Что это за сервис
**TrueStats** — сервис сквозной аналитики и финансового контроля для селлеров на Wildberries и Ozon.

**Основное позиционирование:**
- "Оцифровка бизнеса на маркетплейсах в пару кликов"
- "Принимайте решения исходя из цифр, а не эмоций"
- "Готовая система управления"

### 1.2 Целевая аудитория
- Селлеры с оборотом от 1.5 млн до 30+ млн ₽/мес
- Предприниматели, управляющие несколькими кабинетами (1-10+)
- Средний и крупный бизнес на маркетплейсах

### 1.3 Ключевые функции

**Основные модули:**
1. **Оцифровка** — дашборд с ключевыми показателями, графики, топ-5 товаров по маржинальности, финансовая таблица с ~20 показателями
2. **Себестоимость** — учёт фактической себестоимости по каждой поставке (не средней), динамика изменений
3. **Аналитика по артикулам** — продажи, прибыль, расходы на логистику и хранение, рекламные расходы по размерам
4. **ОПиУ (Отчёт о прибылях и убытках)** — главный инструмент управленческого учёта, показывает финансовый результат за период
5. **Операционные расходы** — учёт расходов, не отражённых в отчётах МП (фото, сервисы, ЗП, аренда, комиссии)
6. **Отчёт по неделям** — сводный отчёт с основными показателями понедельно

**Новинка — модуль РНП:**
- Влияние решений на прибыль в реальном времени
- Точный прогноз маржи по каждому товару
- Анализ продаж, рекламы и воронок

## 2. ОБЗОР НАШЕГО ПРОДУКТА: OzonCheck

### 2.1 Что это за сервис
**OzonCheck** — финансовая аналитика для маркетплейсов Ozon и Wildberries. Расчёт чистой прибыли в 2 клика.

**Основное позиционирование:**
- "Узнайте реальную прибыль на Ozon и Wildberries"
- "Загрузите отчёт — через 30 секунд увидите P&L, расходы и чистую прибыль"
- "Простой инструмент, который заменяет дорогие сервисы аналитики"

### 2.2 Целевая аудитория
- Начинающие и средние селлеры на Ozon и Wildberries
- Предприниматели, которым нужна простая и быстрая аналитика
- Селлеры, ценящие конфиденциальность данных

### 2.3 Ключевые функции

**Основные модули:**
1. **Финансовый дашборд** — выручка, чистая прибыль, маржинальность, заказы с динамикой
2. **P&L отчёт** — каскадная waterfall-диаграмма от выручки до прибыли
3. **Сводный отчёт** — агрегированные показатели с группировкой по неделям/месяцам
4. **Аналитика расходов** — детальная разбивка расходов по категориям (комиссии, логистика, хранение, штрафы, реклама)
5. **Операционные расходы** — учёт постоянных расходов (аренда, ЗП, упаковка, транспорт, маркетинг, ПО)
6. **Себестоимость** — загрузка и управление себестоимостью с версионностью
7. **Остатки и склад** — остатки на складах МП, ABC-классификация запасов
8. **Расчёт налогов** — УСН 6%, 15%, патент, ОСНО с учётом вычетов и взносов
9. **Товарная аналитика и реклама** — детальная таблица по товарам с метриками (расход, ДРР, конверсия)
10. **ABC / XYZ анализ** — классификация товаров по выручке и стабильности спроса (матрица 3×3)
11. **Рекомендации по закупкам** — расчёт что, сколько и когда заказать на основе ABC/XYZ
12. **Unit-экономика** — экономика единицы товара (себестоимость, маржа, GMROI, ROAS, ROI)
13. **ИИ-консультант** — персональные рекомендации на основе анализа данных

**В разработке:**
- Telegram-уведомления (для тарифа Бизнес)
- Автозагрузка по API

**Условия:**
- 3 дня полного доступа бесплатно
- Без банковской карты
- Отмена в один клик
- Первый отчёт — бесплатно

### 2.5 УТП и преимущества
- **Простота** — всё в 2 клика, без перегруза
- **Цена** — дешевле конкурентов (в 2-4 раза)
- **Без API** — не нужно давать доступ к кабинету
- **Скорость** — 30 секунд до результата
- **Конфиденциальность** — данные не отправляются на сервер, всё в браузере
- **Нет лишних инструментов** — только то, что нужно
- **ИИ-консультант** — уникальная фича с персональными рекомендациями

### 2.6 Технические особенности
- Работает без API — загрузка файлов
- Все расчёты в браузере (client-side)
- Данные не хранятся на серверах
- Без лимитов по обороту
