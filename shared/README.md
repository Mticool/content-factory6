# 🧠 Общая память команды

> Эта папка доступна ВСЕМ агентам через `shared/`

## Структура

| Папка | Что хранить | Кто пишет |
|-------|-------------|-----------|
| `research/` | Исследования, кастдев, анализ ЦА, конкуренты | sinsey, analyst |
| `decisions/` | Принятые решения, стратегия, позиционирование | sinsey, marat |
| `knowledge/` | Накопленные знания, инсайты, факты | все |
| `briefs/` | Брифы на задачи, ТЗ между агентами | marat |

## Правила

1. **Пиши сюда** когда результат работы полезен другим агентам
2. **Читай отсюда** перед началом задачи — проверь что уже наработано
3. **Формат файлов:** `YYYY-MM-DD-тема.md`
4. **Не дублируй** — если есть похожий файл, дополни его

## Примеры

- `research/2026-03-06-audience-pain-points.md` — Синсей наресёрчил боли ЦА
- `decisions/2026-03-06-content-strategy-q2.md` — стратегия на квартал
- `knowledge/ozon-commission-changes.md` — факты про комиссии Ozon
- `briefs/2026-03-06-threads-launch.md` — бриф на серию постов
