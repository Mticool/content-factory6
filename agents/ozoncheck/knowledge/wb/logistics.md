## 📦 БАЗОВЫЕ ТАРИФЫ ЛОГИСТИКИ (с 15.09.2025)

### Тарифы зависят от объёма товара:

#### Товары до 1 литра:

| Объём товара | Тариф (FBW/FBS) |
|--------------|-----------------|
| 0,001-0,200 л | **23₽** |
| 0,201-0,400 л | **26₽** |
| 0,401-0,600 л | **29₽** |
| 0,601-0,800 л | **30₽** |
| 0,801-1,000 л | **32₽** |

**Примечание:** Для товаров до 1 литра тарифы **снижены** (ранее были выше).

---

#### Товары свыше 1 литра:

**Формула:** Первый литр + доп. литры

| Компонент | Тариф (FBW/FBS) |
|-----------|-----------------|
| **Первый литр** | **46₽** |
| **Каждый доп. литр** (свыше 1 л) | **+14₽** |

**Примеры:**

**Товар 3 литра:**
- 46₽ + (2 × 14₽) = **74₽**

**Товар 5 литров:**
- 46₽ + (4 × 14₽) = **102₽**

**Товар 10 литров:**
- 46₽ + (9 × 14₽) = **172₽**

## 📊 КОЭФФИЦИЕНТ СКЛАДА (ЛОГИСТИКИ)

**Базовый тариф** умножается на **коэффициент склада**, который зависит от:
- Региона склада
- Загруженности склада
- Направления доставки

### Примеры коэффициентов:

| Регион / Склад | Коэффициент |
|----------------|-------------|
| Москва (центр) | **0,75-1,00** |
| Санкт-Петербург | **0,80-1,00** |
| Екатеринбург | **1,00-1,20** |
| Новосибирск | **1,20-1,50** |
| Краснодар | **1,00-1,30** |
| Дальний Восток | **1,50-2,00** |

**Формула:**
Стоимость логистики = Базовый тариф × Коэффициент склада

**Пример:**
- Товар 5 литров
- Базовый тариф: 102₽
- Коэффициент склада: 1,20
- **Итого:** 102₽ × 1,20 = **122,40₽**

## 📅 ФИКСАЦИЯ ТАРИФОВ

**Важно:** Тариф фиксируется при выборе даты поставки:

### Срок фиксации:

- **90 дней** — для категорий: Одежда, Спортивная одежда, Головные уборы, Обувь, Одежда для малышей, Бельё, Аксессуары для малышей
- **60 дней** — для остальных категорий

**Что фиксируется:**
- Базовый тариф
- Коэффициент склада

**Если товар не продан** за период фиксации — применяются актуальные тарифы на момент продажи.

## 🎯 ИНДЕКС ЛОКАЛИЗАЦИИ (ИЛ) И КТР

С **22.09.2025** Wildberries изменил расчёт индекса локализации.

### Новая система:

**ИЛ** теперь — это **среднее значение КТР** (коэффициента территориального распределения) по всем заказам.

### Таблица КТР (Коэффициент территориального распределения):

| Индекс локализации (ИЛ) | КТР |
|--------------------------|-----|
| ≥75% | **0,50** (скидка -50% на логистику) |
| 70-74% | **0,60** |
| 65-69% | **0,70** |
| 60-64% | **0,80** |
| 55-59% | **0,90** |
| 50-54% | **0,95** |
| 45-49% | **1,00** (базовый тариф) |
| 40-44% | **1,10** |
| 35-39% | **1,20** |
| 30-34% | **1,30** |
| 25-29% | **1,40** |
| 20-24% | **1,50** |
| 15-19% | **1,60** |
| 10-14% | **1,70** |
| 5-9% | **1,80** |
| 0-4% | **2,00** (наценка +100% на логистику) |

**Формула:**
Стоимость логистики = Базовый тариф × Коэфф. склада × **КТР**

**Пример:**
- Товар 5 литров
- Базовый тариф: 102₽
- Коэфф. склада: 1,00
- ИЛ = 40% → КТР = 1,10
- **Итого:** 102₽ × 1,00 × 1,10 = **112,20₽**

## 📏 КОЭФФИЦИЕНТ ГАБАРИТОВ (КГ)

**Что это:** Штрафной коэффициент за **превышение заявленных габаритов** товара.

### Таблица КГ:

| Превышение фактического объёма | КГ |
|--------------------------------|----|
| До 10% | **1,00** (без штрафа) |
| 10-50% | **5,00** |
| Свыше 50% | **10,00** |

**Влияние:**
- КГ умножает стоимость **логистики и хранения**
- Проверяется еженедельно на основе выборочных замеров

**Формула:**
Стоимость логистики = Базовый тариф × Коэфф. склада × КТР × **КГ**

**Пример с превышением габаритов:**
- Товар 5 литров (заявлено), фактически 6 литров (+20%)
- Базовый тариф: 102₽
- КГ = 5 (за превышение 10-50%)
- **Итого:** 102₽ × 1,00 × 1,00 × 5 = **510₽** (вместо 102₽!)

## 🔄 ОБРАТНАЯ ЛОГИСТИКА

### При возврате товара:

**Стоимость обратной логистики:**

#### FBW (со склада WB):
- **50₽** за единицу товара (независимо от объёма)

#### FBS (со склада продавца):
**Формула:** 128₽ за первый литр + 9,5₽ за каждый доп. литр

**Пример (товар 5 литров):**
- 128₽ + (4 × 9,5₽) = **166₽**

**Куда возвращается товар:**
- **FBW:** на склад Wildberries
- **FBS:**
  - В ПВЗ продавца (если настроено)
  - На склад WB (по выбору продавца)

**Автовозврат:** Некоторые категории товаров (шины, крупногабарит) всегда возвращаются в ПВЗ продавца.

## 🧮 ПРИМЕР ПОЛНОГО РАСЧЁТА ЛОГИСТИКИ

**Товар:** Футболка 2000₽, объём 1 литр  
**Схема:** FBW  
**Склад:** Москва (коэфф. 1,00)  
**ИЛ:** 65% → КТР = 0,70  
**Период хранения:** 30 дней  
**Сценарий:** Возврат товара  

### Расходы:

1. **Приёмка:** 1 л × 3,5₽ = **3,50₽**
2. **Прямая логистика:**
   - Базовый тариф (до 1 л): 32₽
   - С учётом коэфф. склада: 32₽ × 1,00 = 32₽
   - С учётом КТР: 32₽ × 0,70 = **22,40₽**
3. **Хранение (30 дней, бесплатно):** **0₽**
4. **Обратная логистика (возврат):** **50₽**

### ИТОГО ЛОГИСТИКА:
**3,50₽ + 22,40₽ + 50₽ = 75,90₽**

## 📉 FBS — ОСОБЕННОСТИ ЛОГИСТИКИ

### Отличия от FBW:

1. **Коэффициент склада** — применяется с 07.08.2025 (как для FBO)
2. **КТР не применяется** — индекс локализации не влияет на FBS
3. **Коэффициент габаритов (КГ)** — применяется (за превышение габаритов)
4. **Обратная логистика** — дороже (128₽ + 9,5₽/л вместо 50₽)
