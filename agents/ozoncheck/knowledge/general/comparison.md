## 🧮 ПОЛНЫЙ ПРИМЕР СРАВНЕНИЯ

**Товар:** Футболка 1500₽, объём 1 литр  
**Схема:** FBO/FBW  
**Сценарий:** Продажа + возврат  

### OZON FBO (до 06.04.2026):

1. **Комиссия (24%):** 1500₽ × 0,24 = **360₽**
2. **Эквайринг (1,5%):** 1500₽ × 0,015 = **22,50₽**
3. **Логистика (СВД 35ч):**
   - Базовая: 46,77₽
   - С учётом СВД (1,320): 46,77₽ × 1,320 = 61,74₽
   - Доп. комиссия (1,60%): 1500₽ × 0,016 = 24₽
   - Доставка до ПВЗ: 25₽
   - **Итого:** 61,74₽ + 24₽ + 25₽ = **110,74₽**
4. **Обратная логистика:** 46,77₽ + 15₽ = **61,77₽**

**ИТОГО РАСХОДОВ:** 360₽ + 22,50₽ + 110,74₽ + 61,77₽ = **555,01₽**

**ПРИБЫЛЬ:** 1500₽ - 555,01₽ = **944,99₽**

---

### WILDBERRIES FBW:

1. **Комиссия (34,5%):** 1500₽ × 0,345 = **517,50₽**
2. **Эквайринг (1,5%):** 1500₽ × 0,015 = **22,50₽**
3. **Комиссия за платёж (1,5%):** 1500₽ × 0,015 = **22,50₽**
4. **Приёмка:** 1 л × 3,5₽ = **3,50₽**
5. **Прямая логистика (КТР 0,70):**
   - Базовая: 32₽
   - С учётом КТР: 32₽ × 0,70 = **22,40₽**
6. **Обратная логистика:** **50₽**

**ИТОГО РАСХОДОВ:** 517,50₽ + 22,50₽ + 22,50₽ + 3,50₽ + 22,40₽ + 50₽ = **638,40₽**

**ПРИБЫЛЬ:** 1500₽ - 638,40₽ = **861,60₽**
