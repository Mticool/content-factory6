## Проблема

Все sub-агенты падают с ошибкой 403 "Verify your account to continue" при запуске через `sessions_spawn`.

## Воспроизведение

```bash
sessions_spawn(agentId="lena", task="Тест")
# Результат: 403 ошибка через 9-20 секунд
```

**Попытки исправления:**
1. ❌ Удаление AIM агентов из конфига — не помогло
2. ❌ Перезапуск OpenClaw — не помогло
3. ❌ Тестирование после перезапуска — та же ошибка

## Гипотезы

1. **Биллинг/лимиты:** OpenClaw блокирует новые сессии (превышен лимит одновременных запросов?)
2. **Права доступа:** Главный агент имеет токен, sub-агенты пытаются создать новый → отказ
3. **Quota:** API провайдеров ограничивают количество сессий

**Главный агент делает всю работу:**
- У marat-m есть 31 скилл (включая swipefile, storytelling, selling-meanings)
- Доказано на практике: 3 видео (137 мин) за 45 минут работы
- Минус: дольше чем делегирование
- Плюс: качество то же, результат гарантирован

1. Лена → аналитические секции (8 шт)

3. Ускорение в 2-3 раза
