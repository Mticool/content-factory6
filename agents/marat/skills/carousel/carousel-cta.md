## Типы CTA

### 1. Follow CTA

**Когда:** Цель — рост подписчиков

**Текст варианты:**
```
"Follow @username for more [тема]"
"Follow me for daily [тема] tips"
"Want more? Follow @username"
"Follow for more valuable content like this"
"Подписывайся, чтобы не пропустить"
```

**Визуал:**
```
┌─────────────────────────────────────┐
│                                     │
│         [ФОТО АВТОРА]               │
│         (круглое, 40%)              │
│                                     │
│         @username                   │
│                                     │
│    "Follow me for more [тема]"      │
│                                     │
│      [❤️] [💬] [📤] [🔖]            │
│                                     │
└─────────────────────────────────────┘
```

**NanoBanana промпт:**
```
Professional headshot of [person from Image1 / character description].
Friendly, approachable expression, warm smile.
Clean, simple background ([solid color/gradient]).
Good lighting, flattering, professional quality.
Shot on portrait lens, shallow depth of field.
Aspect ratio 4:5.

Leave space below photo for:
- @username handle
- "Follow for more" text
- Social media action icons
```

## CTA по типу карусели

| Тип карусели | Лучший CTA | Почему |
|--------------|-----------|--------|
| **Prompt** | Comment "PROMPTS" | Люди хотят полный список |
| **Expert** | Follow + Save | Полезный контент — сохраняют |
| **Minimalist** | Follow | Простой, ненавязчивый |
| **Viral** | Question CTA | Провоцирует обсуждение |
| **Grid** | Save + Link in bio | Много инфы — сохраняют |

## Элементы дизайна CTA

### Плашка с аватаром
```
┌──────────────────────┐
│  [○]  @username      │
│       Follow         │
└──────────────────────┘
```
- Аватар: круглый, 40-60px
- Username: bold
- Кнопка Follow: акцентный цвет

### Иконки действий
```
[❤️ Like] [💬 Comment] [📤 Share] [🔖 Save]
```
- Размер: 24-32px
- Стиль: outline или filled
- Расположение: горизонтально внизу

### Стрелки/указатели
```
↓ Link in bio
→ Swipe for more
↩ Back to slide 1
```

## Best Practices

### DO
- Один чёткий CTA (не больше двух)
- Конкретное действие ("Comment LINK", не "Engage")
- Визуально отличаться от content слайдов
- Включать фото автора (повышает доверие)
- Использовать акцентный цвет для ключевых слов

### DON'T
- Слишком много просьб
- Агрессивный тон ("SUBSCRIBE NOW!!!")
- Мелкий нечитаемый текст
- Перегруженный дизайн
- CTA без объяснения выгоды

## A/B тестирование CTA

### Что тестировать
1. **Тип CTA:** Follow vs Comment vs Save
2. **Текст:** Разные формулировки
3. **С фото vs без фото**
4. **Цвет акцента**
5. **Позиция элементов**

### Метрики
- Follow CTA → прирост подписчиков
- Comment CTA → количество комментариев
- Save CTA → saves в insights
- Link CTA → клики по ссылке в bio

### Формула
```
Лучший CTA = Много тестов + Анализ данных + Итерация
```
