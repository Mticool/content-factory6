# YouTube Скрипт-Агент

Универсальный инструмент для создания YouTube контента: от идеи до публикации.

**Принцип успеха:** Хук = 50% успеха видео. Первые 30 секунд решают всё.

## Быстрый старт

Создание полного скрипта с хуком:

```bash
# Полный скрипт по теме
"Напиши youtube скрипт на тему: [ТЕМА]"

# Только хук для видео
"Создай хук для youtube видео про [ТЕМА]"

# Структура видео с пейоффами
"Сделай структуру youtube видео про [ТЕМА]"
```

## Параметры/Настройки

| Формат | Длительность | Объём скрипта | Назначение |
|--------|-------------|---------------|------------|
| Shorts | До 60 сек | 100-150 слов | Вирусность, охваты |
| Short | 2-5 мин | 300-750 слов | Быстрые советы |
| Medium | 6-12 мин | 900-1800 слов | Обучающие ролики |
| Long | 15+ мин | 2250+ слов | Глубокий разбор |

**Формула времени:** 150 слов ≈ 1 минута речи

## Примеры использования

### Создание скрипта
```
Напиши youtube скрипт на 8 минут про "5 ошибок начинающих предпринимателей"
```

### Аудит хука  
```
Проверь мой хук по ВИСП: "Сегодня расскажу про ошибки в бизнесе"
```

### SEO оптимизация
```
Создай заголовок, описание и теги для видео про тайм-менеджмент
```

### Превью концепт
```
Придумай концепт thumbnail для видео "Как я заработал первый миллион"
```

## Частые ошибки

❌ **Слабый хук** — не прошёл проверку ВИСП  
❌ **"Вы все" вместо "ты"** — говори с одним человеком  
❌ **Нет открытых петель** — зрители уходят  
❌ **Пейоффы не награждают** — каждый должен давать ценность  
❌ **Мега-пейофф не в конце** — самое важное должно быть последним  
❌ **Неестественный CTA** — должен быть органичным  
❌ **Игнорирование сброса внимания** — каждые 3-5 минут нужна встряска

### Ключевые фреймворки

**ВИСП (для хука):**
- **В**ыгода — что получит зритель?
- **И**нтрига — есть ли вопрос в голове?
- **С**рочность — почему не отложить?
- **П**ерсонализация — это про меня?

**TTS (для обучающего контента):**
- **T**arget — кто аудитория
- **T**ransformation — что изменится  
- **S**takes — что поставлено на карту

**Структура пейоффа:**
```
Завязка → Развитие → "И что?" → Зингер → Открытая петля
```
