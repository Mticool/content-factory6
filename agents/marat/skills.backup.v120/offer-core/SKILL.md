# Offer Core

> Ядро оффера = одна фраза, после которой все поворачиваются с вопросом "ЧТО?!"

## Что это

**Ядро оффера** — основа, на которой строится ВСЁ:
- Продажники для tripwire
- Продажники для основного продукта
- Контент
- Реклама
- Прогревы

Не путать с полным оффером! Оффер = большая страница с обоснованиями. 
Ядро = одна мощная фраза.

## Главная формула

```
Результативный глагол + Идеальный результат + Скорость достижения + [Технология]
```

### Компоненты

**Результативный глагол:**
- Зарабатывать
- Научиться
- Освоить
- Похудеть
- Получить

**Идеальный результат:**
- Критериально измеримый (цифры) ИЛИ
- Понятный через состояние

**Скорость:**
- За X недель/месяцев
- Всего за X дней

**Технология (опционально):**
- Освоив технологию [название]
- При помощи [метод]
- Используя [инструмент]

## Тест "Конференция"

```
Представь: ты на конференции по твоей теме.
Произносишь свою фразу.
Все поворачиваются с реакцией "ЧТО?!"
→ Ядро работает.

Никто не поворачивается.
→ Ядро слабое, переделать.
```

## Ключевые желания vs Обычные

**Ключевые** — то, ради чего люди РЕАЛЬНО приходят:
- Зарабатывать много
- Тратить времени мало
- Получить конкретный результат

**НЕ ключевые** — второстепенные:
- "Настраивать без банов"
- "Иметь упакованный профиль"
- "Получить сертификат"

> Люди думают ТОЛЬКО О СЕБЕ. Ключевые желания = про их личную выгоду.

## Продуктология (из разбора запуска на 59.6 млн)

### Качественный продукт — это тот, который:

- Даёт результат (рост % кейсов)
- Повышает средний чек через ценность
- Повышает LTV (ученики идут из продукта в продукт)
- Генерирует отзывы → рост продаж
- Высокий NPS → ученики = фанаты

### Рост % кейсов = рост всего

```
↑ Кейсы → ↑ Конверсия → ↑ Выручка → Сильный бренд → Повторные покупки (экономия на трафике)
```

### Что влияет на % кейсов

- ЦА (новички vs профи — разные подходы)
- Проработанность программы (методология)
- Обратная связь (кураторы, наставники)
- Комьюнити (людям одиноко)
- Прогретость аудитории

### Для новичков vs профи

| | Новички | Профи |
|---|---|---|
| Проблема | Пропасть между знанием и действием | Нужна суть без воды |
| Решение | Поддержка, психолог, бадди, десятки, развёрнутая ОС | Ключевые знания → действия |
| Доп. активности | Модули по барьерам, отработка страхов | Минимум, фокус на результат |

### НЕ перегружать!

> Лучший способ убить доходимость = добавить всю ценность и даже больше

Выходы:
- База знаний
- Бонусные уроки за сдачу ДЗ
- Деление на обязательные/необязательные блоки
- Уровни сложности
- Живые НЕ обучающие форматы (мастермайнды)

### Структура тарифов

- Тариф С участием эксперта (дорогой)
- Тариф БЕЗ участия эксперта (сильные наставники, записи лекций)
- Масштабирование: наставники → старший наставник → самоуправление

### Методолог (ключевая роль)

- Проектирует программу с пониманием влияния на продажи и кейсы
- ЗП: 80-300к/мес
- НЕ берите того, кто просто делает что скажете
- Должен понимать взаимосвязь продаж и продукта

### NPS (как измерять)

- Массовый продукт: стоп-урок перед модулем, упор на "выбери ответ"
- Наставничество: промежуточная NPS на середине
- NPS не главное — важен результат ученика

### Фишки продукта

- Мотивационный живой эфир
- Служба заботы обзванивает выпавших
- Внешняя мотивация за кейс (подарок, пиар, сессия)
- Подружить участников друг с другом
- Трекер в прямой коммуникации с участником
- Юмор (мемы, ММ на поржать)
- Практические задания с вау-эффектом (быстрый результат)
- Оффлайн встречи

### Договор оферты

- Если запуск от 1 млн — обязательно через юриста
- Защищает от возвратов и неадекватных участников

## Правила из линз мировых экспертов

### Seth Godin (Small Audience)
- 2000 охватов → 59.6 млн. Не гнаться за охватами. Глубина прогрева × качество > количество

### Jay Abraham (Partnerships)
- Рекомендации лидеров ниши = бесплатный целевой трафик (Host-Parasite strategy)
- Формализовать реферальную систему: ученик→друг→бонус обоим. У Егора 25% выручки = рекомендации, но без системы это случайность
