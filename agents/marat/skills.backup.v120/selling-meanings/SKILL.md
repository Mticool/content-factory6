# Продающие Смыслы

Система извлечения продающих смыслов из экспертного контента, кейсов и исследований аудитории.

## Быстрый старт

```bash
# Извлечь смыслы из текста
"Извлеки продающие смыслы из: [текст/транскрипт]"

# Упаковать смысл в пост
"Упакуй смысл '{смысл}' в threads пост"

# Карта смыслов для продукта
"Составь карту смыслов для продукта: [название]"
```

## Что такое продающий смысл

Продающий смысл — это идея, которая:
1. Резонирует с болью/желанием аудитории
2. Позиционирует {ваш эксперт} как решение
3. Может быть упакована в разные форматы (пост, рилс, сторис, лендинг)

## Источники смыслов

| Источник | Как извлекать |
|----------|---------------|
| Кейсы клиентов | Результат + эмоция до/после |
| Возражения | Перевернуть в аргумент |
| FAQ | Частые вопросы = боли |
| Отзывы | Цитаты с конкретикой |
| Конкуренты | Что они НЕ говорят |
| Личный опыт | Истории трансформации |

## Формула смысла

```
СМЫСЛ = Боль аудитории + Инсайт эксперта + Доказательство
```

### Пример
- **Боль:** "Не могу продавать, чувствую себя впаривателем"
- **Инсайт:** "Продажа = помощь, если продукт реально решает проблему"
- **Доказательство:** "Клиент пришёл со словами 'наконец кто-то объяснил нормально'"
- **Смысл:** Продавать — значит помогать найти решение, которое человек уже ищет

## Карта смыслов

Для каждого продукта составляй карту:

1. **Смыслы входа** — почему начать сейчас
2. **Смыслы доверия** — почему именно {ваш эксперт}
3. **Смыслы результата** — что получит клиент
4. **Смыслы срочности** — почему не ждать
5. **Смыслы отстройки** — чем отличается от конкурентов

## Упаковка смыслов

Один смысл → множество форматов:

| Формат | Как упаковать |
|--------|---------------|
| Threads пост | Хук из смысла + раскрытие в 500 символов |
| Reels | Визуальная метафора смысла |
| Сторис | Опрос/вопрос вокруг смысла |
| Лендинг | Блок с заголовком из смысла |
| Email | Тема письма из смысла |

## Частые ошибки

❌ Смысл без конкретики — "мы помогаем расти" (кому? как? на сколько?)
❌ Смысл-клише — "качество по доступной цене"
❌ Смысл без доказательства — голословное утверждение
❌ Один смысл на всё — нужна карта из 15-20 смыслов

## Карта смыслов из кастдевов (метод Егора Пырикова)

### Процесс создания

```
Распаковка эксперта + Анализ аудитории + Кастдевы
                    ↓
             КАРТА СМЫСЛОВ (20-30 штук)
                    ↓
    Стратегия прогрева → Контент → Веб → Продажи
```

### Кастдевы → Смыслы

1. Провести 2-3 созвона на каждый сегмент ЦА
2. Выписать ВСЕ боли, желания, возражения
3. Добавить боли околоцелевой аудитории (расширяет точность)
4. На каждую боль → способ закрытия (кейс/продукт/опыт)
5. Результат: банк из 20-30 смыслов

### Формат записи

```
БОЛЬ: [конкретная боль из кастдева]
ЗАКРЫТИЕ: [как продукт/эксперт решает]
ДОКАЗАТЕЛЬСТВО: [кейс/цифра/факт]
ФОРМАТ: [сторис/пост/эфир/подкаст]
```

### Источники смыслов (приоритет)

1. **Кастдевы** — первичный источник (обновлять перед каждым запуском!)
2. **Сообщения из Директа** — живые запросы и формулировки ЦА
3. **Анкеты предзаписи** — вопрос "ваш запрос на продукт"
4. **Эфиры и разборы** — вопросы участников
5. **Конкуренты** — что они НЕ говорят

### Правило: смыслы меняются

> Кастдевы перед КАЖДЫМ запуском. Тренды меняются, рынок меняется, появляются новые боли. Глаз замыливается.

### Связка "Боль + Кейс"

Самая эффективная единица контента:
```
Сегодня подняли боль/проблему/возражение
→ Закрыли в конце дня кейсом
```

Кейсы под разные сегменты ЦА. Через день начиная со 2-й недели прогрева.

## Правила из линз мировых экспертов

### Eugene Schwartz (5 уровней осведомлённости)
- Контент должен покрывать ВСЕ 5 уровней: Unaware → Problem Aware → Solution Aware → Product Aware → Most Aware
- Unaware: "Почему вы застряли". Problem: раскрытие боли. Solution: "есть система". Product: кейсы+уникальность. Most Aware: прямой оффер
- Заголовок определяется уровнем осведомлённости аудитории

### Seth Godin (Tribe Marketing)
- Продавать принадлежность к сообществу, не обучение. "Люди как мы делают так" > "Научу вас"
- Анкета предзаписи = permission marketing. Человек сам просит продать ему
- 2000 охватов → 59.6 млн. Глубина × качество аудитории > количество
