# Поиск товаров на маркетплейсах

Автоматический поиск товаров с получением ссылок.

## Поддерживаемые платформы

- ✅ **Авито** — объявления
- ✅ **Ozon** — товары
- ✅ **Wildberries** — товары

## Использование

```bash
./search-products.sh <маркетплейс> <запрос> [количество]
```

### Примеры

**Авито:**
```bash
./search-products.sh avito 'игрушки трансформеры' 10
./search-products.sh avito 'компьютер' 20
./search-products.sh avito 'iphone 15' 15
```

**Ozon:**
```bash
./search-products.sh ozon 'чехол iphone' 20
./search-products.sh ozon 'наушники bluetooth' 15
./search-products.sh ozon 'power bank' 10
```

**Wildberries:**
```bash
./search-products.sh wb 'футболка мужская' 20
./search-products.sh wb 'кроссовки nike' 15
./search-products.sh wb 'рюкзак' 10
```

## Workflow: Поиск → Парсинг → Анализ

### Шаг 1: Поиск товаров
```bash
./search-products.sh avito 'трансформеры' 10
```
Получаем файл: `/tmp/search_avito_трансформеры_urls.txt`

### Шаг 2: Массовый парсинг найденных товаров
```bash
./batch-scrape.sh /tmp/search_avito_трансформеры_urls.txt
```
Получаем подробные данные по всем 10 товарам.

### Шаг 3: Анализ
- Сравнение цен
- Анализ описаний
- Мониторинг конкурентов

## Параметры по умолчанию

| Параметр | Значение |
|----------|----------|
| Количество результатов | 20 |
| Регион (Авито) | Москва |
| Таймаут | 90 секунд |

## Ограничения

**Безопасно:**
- 1-2 поиска в час
- До 20 результатов за раз

**Рискованно:**
- Частые поиски (каждые 5 минут)
- 50+ результатов

## Что возвращается

### Авито
- Название
- Цена
- URL
- Локация
- Дата публикации
- Изображение

### Ozon
- Название
- Цена
- URL
- Рейтинг
- Изображение

### Wildberries
- Название
- Цена (если доступна)
- URL
- Изображение

## Практические примеры

### Мониторинг цен конкурентов
```bash
# Поиск похожих товаров
./search-products.sh ozon 'power bank 20000' 15

# Парсинг всех найденных
./batch-scrape.sh /tmp/search_ozon_power_bank_20000_urls.txt
```

### Поиск поставщиков на Авито
```bash
./search-products.sh avito 'оптом игрушки' 20
```

### Анализ ниши на Wildberries
```bash
./search-products.sh wb 'органайзер для обуви' 20
```

---

**Совет:** Комбинируй поиск + batch-парсинг для быстрого анализа рынка.
