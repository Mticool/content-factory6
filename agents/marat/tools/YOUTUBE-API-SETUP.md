# YouTube Data API — Получение ключа

> Бесплатно, 10,000 запросов/день, займёт 5 минут

---

## Шаг 1: Создать Google Cloud проект

1. **Открой:** https://console.cloud.google.com/
2. **Войди** через свой Google аккаунт (mti2324@gmail.com или другой)
3. **Создай новый проект:**
   - Нажми на выпадающий список проектов (вверху слева)
   - Кнопка **"New Project"** (Создать проект)
   - Имя проекта: `OzonCheck YouTube Monitor`
   - Жми **"Create"** (Создать)

---

## Шаг 2: Включить YouTube Data API v3

1. **Открой меню** (☰ слева вверху)
2. Перейди: **APIs & Services** → **Library**
3. **Найди:** В поиске введи `YouTube Data API v3`
4. **Открой** YouTube Data API v3
5. Нажми **"Enable"** (Включить)

---

## Шаг 3: Создать API ключ

1. **Открой меню** (☰ слева вверху)
2. Перейди: **APIs & Services** → **Credentials** (Учётные данные)
3. Нажми **"+ CREATE CREDENTIALS"** (вверху)
4. Выбери **"API key"**
5. **Скопируй ключ** (длинная строка типа `AIzaSyC...`)

⚠️ **Важно:** Ключ показывается только один раз — сразу скопируй!

---

## Шаг 4: Ограничить ключ (безопасность)

1. После создания нажми **"RESTRICT KEY"**
2. **Application restrictions:**
   - Выбери: **"IP addresses"**
   - Добавь IP твоего сервера: `YOUR_SERVER_IP` (можно узнать командой `curl ifconfig.me`)
   - Или оставь **"None"** (если не критично)

3. **API restrictions:**
   - Выбери: **"Restrict key"**
   - Отметь только: **YouTube Data API v3**

4. Нажми **"Save"**

---

## Шаг 5: Отправить мне ключ

Скопируй API ключ и пришли мне:

```
AIzaSy...вся_длинная_строка
```

Я его сохраню в защищённый файл (`tools/youtube-api-key.txt`, chmod 600) и сразу запущу мониторинг!

## Возможные проблемы

### "API key not found"
- Подожди 1-2 минуты после создания ключа
- Обнови страницу

### "Billing account required"
- Google может запросить привязку карты (не снимают деньги)
- Бесплатный лимит: 10,000 запросов/день = ~300 проверок видео
- Мы будем использовать ~100 запросов/неделю

### Нужна помощь?
- Скриншот экрана → я подскажу что нажимать
- Или просто отправь проблему — разберёмся
