# Batch Scraper — Массовый парсинг товаров

Парсинг 10-15 товаров с Ozon, Wildberries и Avito с защитой от блокировки.

## Возможности

✅ Парсит Ozon и Wildberries  
✅ Автоматические задержки (20-45 сек между товарами)  
✅ Длинные паузы каждые 5 товаров (3-5 минут)  
✅ Ротация User-Agent  
✅ Имитация поведения пользователя (прокрутка, клики)  
✅ Мониторинг блокировки  
✅ Экспорт в JSON  

## Использование

### Вариант 1: Файл с URL

Создай `urls.txt`:
```
https://ozon.ru/t/abc123
https://www.wildberries.ru/catalog/12345/detail.aspx
https://ozon.ru/t/xyz789
```

Запусти:
```bash
./batch-scrape.sh urls.txt
```

### Вариант 2: URL в командной строке

```bash
./batch-scrape.sh \
  "https://ozon.ru/t/abc123" \
  "https://www.wildberries.ru/catalog/12345/detail.aspx" \
  "https://ozon.ru/t/xyz789"
```

## Защита от блокировки

### Задержки
- **Между товарами:** 20-45 секунд (рандомно)
- **Каждые 5 товаров:** 3-5 минут (длинная пауза)
- **При блокировке:** автоматическая пауза 2 минуты

### Имитация пользователя
- Рандомная прокрутка (3-6 раз по 400-800px)
- "Чтение" страницы (3-7 секунд)
- Смена User-Agent каждые 3-5 запросов

### Ротация User-Agent
Автоматическая смена между:
- Chrome (Windows, Mac, Linux)
- Firefox
- Safari

## Время выполнения

| Товаров | Время (примерно) |
|---------|------------------|
| 5       | 3-5 минут        |
| 10      | 8-12 минут       |
| 15      | 15-20 минут      |

JSON-файл в `/tmp/batch_scrape_YYYYMMDD_HHMMSS.json`:

```json

    "images": ["https://..."],

  },

```

## Что делать при блокировке

Если видишь `❌ БЛОКИРОВКА ОБНАРУЖЕНА!`:

1. Скрипт автоматически увеличит паузу до 2 минут
2. Если блокировки продолжаются — прерви и запусти позже
3. Можно уменьшить количество товаров (5-7 за раз безопаснее)

## Ограничения

**Безопасно:**
- До 15 товаров за раз
- 1-2 запуска в день

**Рискованно:**
- 20+ товаров подряд
- Частые запуски (каждый час)

## Примеры

### Парсинг конкурентов (5 товаров)
```bash
./batch-scrape.sh competitor-urls.txt
```

### Мониторинг цен (10 товаров)
```bash
./batch-scrape.sh price-monitoring.txt
```

### Анализ ниши (15 товаров)
```bash

```
