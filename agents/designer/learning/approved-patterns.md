## 2026-03-06: Стиль карусели — минимализм + золотые акценты

**Что сработало:** Дизайн-система для Instagram каруселей с чередованием светлых/темных слайдов

**Ключевые элементы:**

### Цвета
- **Основа**: черный (#000000) + белый (#FFFFFF)
- **Акцент**: золотой/оранжевый (#D4A942, #E5A836) — для ключевых фраз
- **Фон светлых слайдов**: бежевый/серый (#E8E5DE, #F5F5F5)
- **Фон темных слайдов**: глубокий черный

### Типографика
- **Заголовки**: сжатый жирный шрифт без засечек (condensed bold, uppercase)
- **Тело текста**: легкий читаемый шрифт, ~16-18pt
- **Акценты**: золотой цвет на ключевых словах/фразах

### Брендинг
- **Звездочка (*)**: золотая, левый верхний угол — фирменный знак
- **Круглый логотип**: для персональных обложек (единорог)

### Композиция
- **Воздух**: много белого пространства, минимализм
- **Иерархия**: заголовок (крупно) → описание (средне) → акцент (золотой)
- **Формы**: закругленные углы у карточек (~20-30px radius)
- **Тени**: мягкие тени у карточек на темном фоне

### Форматы слайдов

**1. Текстовые (светлый фон)**
- Бежевый/серый фон
- Звездочка слева сверху
- Заголовок черным, крупно
- Описание мелким
- Ключевая фраза золотом

**2. Карточки (темный фон)**
- Черный фон
- Белая карточка с закругленными углами
- Эффект слоев/теней
- Золотые закладки или акценты

**3. Контрастные блоки**
- Черный фон
- Речевые пузыри: белый + золотой
- Диалоговая структура

**4. С фото**
- Черно-белые портреты
- Серый текстурный или черный фон
- Крупная типографика
- Логотип + навигация (стрелка)

**5. Обложки (стиль журнала)**
- Ч/б фото знаменитостей
- Крупный текст с гранжевым эффектом
- Логотипы брендов
- Яркие элементы (красная лента)

### Текстовая структура
- **Заголовок**: короткий, ёмкий, 2-4 слова
- **Описание**: 1-2 предложения, конкретика
- **CTA/вывод**: выделен золотым

### Чередование
- Светлый → темный → светлый
- Текст → фото → текст
- Разнообразие форматов удерживает внимание

**Почему работает:**
- Высокий контраст — легко читать
- Золотой акцент направляет внимание
- Минимализм — не перегружено
- Профессиональный вид
- Узнаваемый бренд (звездочка)

**RULE:** [design/carousel] — Использовать черно-бело-золотую схему с минимализмом и звездочкой как брендовым элементом. Чередовать светлые/темные слайды. Золотой цвет ТОЛЬКО для ключевых фраз.
