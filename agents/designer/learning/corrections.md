## 2026-03-06: Карусель — консистентность выравнивания

**CORRECTION:** Текст на слайдах скачет — то слева, то по центру

**REASON:** Нет единообразия в композиции → глаз спотыкается при просмотре

**CORRECT:** Все слайды должны иметь одинаковое выравнивание текста. Для текстовых каруселей предпочтительнее **выравнивание слева** — легче читать, профессиональнее.

**RULE:** [design/carousel] — В одной карусели использовать ТОЛЬКО одно выравнивание для всех слайдов. Для длинных текстов — слева. Для коротких хуков/цифр — можно по центру, но тогда ВСЕ слайды по центру.
